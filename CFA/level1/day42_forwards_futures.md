## 📖 1. ফরওয়ার্ড চুক্তি (Forward Contract)

**ফরওয়ার্ড:** দুই পক্ষের মধ্যে একটি চুক্তি — ভবিষ্যতে একটি নির্দিষ্ট তারিখে নির্দিষ্ট দামে একটি সম্পদ কেনা/বেচা।

| বৈশিষ্ট্য | বিবরণ |
|-----------|--------|
| **OTC (Over-the-Counter)** | এক্সচেঞ্জে নয়, সরাসরি দুই পক্ষের মধ্যে |
| **কাস্টমাইজড** | পরিমাণ, তারিখ, দাম — সবকিছু নিজেদের মতো |
| **কাউন্টারপার্টি ঝুঁকি** | অন্য পক্ষ ডিফল্ট করলে ক্ষতি |
| **কোন অগ্রিম পেমেন্ট নেই** | চুক্তির সময় কোনো টাকা লেনদেন হয় না |
| **মেয়াদ শেষে সেটলমেন্ট** | একবারেই সব লেনদেন |

**ফরওয়ার্ড চুক্তির পে-অফ:**

**লং পজিশন (ক্রেতা):** Payoff = S_T - F_0
**শর্ট পজিশন (বিক্রেতা):** Payoff = F_0 - S_T

যেখানে: S_T = মেয়াদে স্পট দাম, F_0 = ফরওয়ার্ড দাম

**উদাহরণ — একজন কৃষক:**
> একজন কৃষক ৩ মাস পর ১০০০ বুশেল গম বিক্রি করবেন। বর্তমান দাম $৫/বুশেল। তিনি একটি ফরওয়ার্ড চুক্তি করেন $৫.২০/বুশেলে (৩ মাস পর ডেলিভারি)।
> - ক্রেতা (মিল): লং পজিশন
> - কৃষক (বিক্রেতা): শর্ট পজিশন
> 
> দৃশ্যকল্প ১: ৩ মাস পর দাম = $৬.০০
> - ক্রেতা লাভ: $৬.০০ - $৫.২০ = **$০.৮০/বুশেল** ✅
> - কৃষকের ক্ষতি: $৫.২০ - $৬.০০ = **-$০.৮০/বুশেল** ❌
> 
> দৃশ্যকল্প ২: ৩ মাস পর দাম = $৪.০০
> - ক্রেতার ক্ষতি: $৪.০০ - $৫.২০ = **-$১.২০/বুশেল** ❌
> - কৃষক লাভ: $৫.২০ - $৪.০০ = **$১.২০/বুশেল** ✅ (দাম কমলেও তিনি বেশি দামে বিক্রি করতে পেরেছেন)

**ফরওয়ার্ডের সুবিধা:**
1. হেজিং — দামের ঝুঁকি কমানো
2. কাস্টমাইজেশন — আপনার প্রয়োজন অনুযায়ী
3. কোন অগ্রিম খরচ নেই

**ফরওয়ার্ডের অসুবিধা:**
1. কাউন্টারপার্টি ঝুঁকি
2. তরলতা কম — সহজে বিক্রি করা যায় না
3. নিয়ন্ত্রণ নেই

## 📖 2. ফিউচার চুক্তি (Futures Contract)

**ফিউচার:** ফরওয়ার্ডের মতোই, কিন্তু এক্সচেঞ্জে ট্রেড হয় এবং স্ট্যান্ডার্ডাইজড।

| বৈশিষ্ট্য | বিবরণ |
|-----------|--------|
| **এক্সচেঞ্জ-ট্রেডেড** | NYSE, CME ইত্যাদিতে |
| **স্ট্যান্ডার্ডাইজড** | পরিমাণ, তারিখ, গুণগত মান নির্দিষ্ট |
| **ক্লিয়ারিং হাউস** | কাউন্টারপার্টি ঝুঁকি নেই |
| **মার্জিন প্রয়োজন** | জামানত দিতে হয় |
| **Mark-to-Market** | প্রতিদিন লাভ/ক্ষতি গণনা |
| **তরল** | সহজেই কেনা-বেচা যায় |

### ফিউচার মার্জিন

| মার্জিনের ধরন | বিবরণ |
|--------------|--------|
| **ইনিশিয়াল মার্জিন** | চুক্তি খোলার সময় জমা দিতে হয় |
| **মেইনটেন্যান্স মার্জিন** | সর্বনিম্ন মার্জিন স্তর বজায় রাখতে হবে |
| **ভ্যারিয়েশন মার্জিন** | ক্ষতি পূরণে অতিরিক্ত মার্জিন দরকার হলে |

**Mark-to-Market প্রক্রিয়া:**
```
দিন ১: ফিউচার প্রাইস $১০০ → $১০৫ (পজিশন +$৫ মার্জিনে যোগ)
দিন ২: ফিউচার প্রাইস $১০৫ → $৯৮ (পজিশন -$৭ মার্জিন থেকে কেটে নেওয়া)
দিন ৩: ফিউচার প্রাইস $৯৮ → $১০২ (পজিশন +$৪ মার্জিনে যোগ)
```

**উদাহরণ — ফিউচার ট্রেডিং:**
> আপনি S&P 500 ফিউচার কিনলেন (লং) ৪,০০০ পয়েন্টে। প্রতিটি পয়েন্ট = $৫০।
> 
> দিন ১: ফিউচার = ৪,০২০ → লাভ = ২০ × $৫০ = **+$১,০০০**
> দিন ২: ফিউচার = ৩,৯৮০ → ক্ষতি = ৪০ × $৫০ = **-$২,০০০**
> দিন ৩: ফিউচার = ৪,০৫০ → লাভ = ৭০ × $৫০ = **+$৩,৫০০**
> নেট = $১,০০০ - $২,০০০ + $৩,৫০০ = **+$২,৫০০**

## 📖 4. ফিউচার প্রাইসিং

ফিউচারের দাম সাধারণত স্পট দামের উপর ভিত্তি করে নির্ধারিত হয়:

**F₀ = S₀ × (1 + r - c)ᵀ**

যেখানে:
- F₀ = ফিউচার প্রাইস
- S₀ = বর্তমান স্পট প্রাইস
- r = রিস্ক-ফ্রি রেট
- c = কনভিনিয়েন্স ইল্ড/স্টোরেজ খরচ
- T = সময়

**বেসিস (Basis):** Basis = Spot Price - Futures Price

**উদাহরণ — প্রাইসিং:**
> সোনার বর্তমান দাম $১,৮০০/আউন্স। ৬ মাসের ফিউচার। রিস্ক-ফ্রি রেট = ৫%, স্টোরেজ খরচ = ১%।
> F = $১,৮০০ × (১ + ০.০৫ - ০.০১)^(০.৫)
> F = $১,৮০০ × (১.০৪)^(০.৫)
> F = $১,৮০০ × ১.০১৯৮ = **$১,৮৩৫.৬৪**

## 📖 5. ফিউচারের ব্যবহার

| ব্যবহার | ব্যাখ্যা |
|---------|----------|
| **হেজিং** | দামের ঝুঁকি কমানো |
| **স্পেকুলেশন** | দামের পরিবর্তন থেকে মুনাফা |
| **আরবিট্রেজ** | বাজারের অদক্ষতা থেকে লাভ |
| **প্রাইস ডিসকভারি** | ভবিষ্যতের দামের ইঙ্গিত |

**হেজিং উদাহরণ — এয়ারলাইন:**
> একটি এয়ারলাইন জ্বালানি তেলের দাম নিয়ে চিন্তিত। বর্তমান দাম $৭০/ব্যারেল। তারা ১ বছর পর ১০০,০০০ ব্যারেল জ্বালানির ফিউচার কিনল $৭২/ব্যারেলে।
> 
> দৃশ্যকল্প: দাম বেড়ে $৮৫/ব্যারেল
> - ফিউচার লাভ: ($৮৫ - $৭২) × ১০০,০০০ = **+$১.৩M**
> - বাস্তবে জ্বালানি কিনতে বেশি খরচ: ($৮৫ - $৭০) × ১০০,০০০ = **-$১.৫M**
> - নেট এফেক্ট: $১.৩M - $১.৫M = **-$০.২M** (ফিউচার না থাকলে ক্ষতি হতো $১.৫M)

## ✍️ অনুশীলন

### প্রশ্ন ১:
একটি ফরওয়ার্ড চুক্তিতে লং পজিশনের পে-অফ কী?

**সমাধান:**
Payoff = S_T - F_0 (মেয়াদে স্পট দাম - ফরওয়ার্ড দাম)

### প্রশ্ন ২:
ফিউচার ও ফরওয়ার্ডের মধ্যে দুটি গুরুত্বপূর্ণ পার্থক্য বলুন।

**সমাধান:**
1. ফিউচার এক্সচেঞ্জে ট্রেড হয়, ফরওয়ার্ড OTC
2. ফিউচার প্রতিদিন Mark-to-Market হয়, ফরওয়ার্ড মেয়াদ শেষে

### প্রশ্ন ৩:
একজন কৃষক ৫,০০০ বুশেল গমের ফিউচার $৪.৫০/বুশেলে শর্ট করলেন। মেয়াদে দাম $৫.২০ হলে তার লাভ/ক্ষতি কত?

**সমাধান:**
শর্ট পজিশন পে-অফ = F_0 - S_T = $৪.৫০ - $৫.২০ = -$০.৭০/বুশেল
মোট ক্ষতি = -$০.৭০ × ৫,০০০ = **-$৩,৫০০** ❌
