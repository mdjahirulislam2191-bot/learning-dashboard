## 📖 1. সম্ভাবনার মৌলিক ধারণা

### সম্ভাবনার নিয়ম
- যে কোনো ঘটনার সম্ভাবনা ০ থেকে ১ এর মধ্যে
- সব ঘটনার সম্ভাবনার সমষ্টি = ১
- P(A না ঘটার) = 1 - P(A)

### শর্তাধীন সম্ভাবনা (Conditional Probability)
P(A|B) = P(A∩B) / P(B)

**উদাহরণ:**
> একটি স্টকের দাম বাড়ার সম্ভাবনা ৬০%। যদি GDP বৃদ্ধি পায়, তাহলে স্টকের দাম বাড়ার সম্ভাবনা ৮০%।

### মোট সম্ভাবনার সূত্র (Total Probability Rule)
P(A) = P(A|B) × P(B) + P(A|B') × P(B')

### বেইস সূত্র (Bayes' Formula)
P(A|B) = P(B|A) × P(A) / P(B)

**ফাইন্যান্সে ব্যবহার:**
> একটি কোম্পানি ডিফল্ট হওয়ার পূর্বাভাস দিতে — নতুন তথ্যের ভিত্তিতে সম্ভাবনা আপডেট করা।

## 📖 2. র্যান্ডম ভ্যারিয়েবল

### ডিসক্রিট (Discrete)
নির্দিষ্ট মান নেয় (যেমন: পাশার সংখ্যা ১-৬)।

### কন্টিনিউয়াস (Continuous)
যেকোনো মান নিতে পারে (যেমন: স্টকের রিটার্ন -∞ থেকে +∞)।

### প্রত্যাশিত মান (Expected Value)
E(X) = Σ [x × P(x)]

**উদাহরণ:**
> একটি বিনিয়োগের তিনটি সম্ভাব্য ফলাফল:
> - লাভ $১০০ (৩০% সম্ভাবনা)
> - লাভ $৫০ (৫০% সম্ভাবনা)
> - ক্ষতি $২০ (২০% সম্ভাবনা)
>
> E(X) = (১০০ × ০.৩০) + (৫০ × ০.৫০) + (-২০ × ০.২০)
> E(X) = ৩০ + ২৫ - ৪ = **$৫১**

### ভ্যারিয়েন্স
Var(X) = E[(X - μ)²] = Σ[(x - μ)² × P(x)]

## 📖 3. Binomial Distribution

দুটি সম্ভাব্য ফলাফল (সফল/ব্যর্থ) সহ স্বাধীন পরীক্ষা।

**সূত্র:** P(X=k) = C(n,k) × p^k × (1-p)^(n-k)

**যেখানে:**
- n = পরীক্ষার সংখ্যা
- k = সফলতার সংখ্যা
- p = সফলতার সম্ভাবনা

**উদাহরণ:**
> একটি ফেয়ার কয়েন ৫ বার টস করলে ৩ বার Head আসার সম্ভাবনা কত?
>
> P(X=৩) = C(৫,৩) × (০.৫)³ × (০.৫)²
> = ১০ × ০.১২৫ × ০.২৫
> = **৩১.২৫%**

**ফাইন্যান্সে ব্যবহার:**
> একটি খারাপ বছর (Bear Market) আসার সম্ভাবনা ৪০%। পরবর্তী ৫ বছরের মধ্যে ৩টি খারাপ বছর আসার সম্ভাবনা কত?

## 📖 4. Normal Distribution

সবচেয়ে গুরুত্বপূর্ণ বিতরণ — বেল আকৃতির।

**বৈশিষ্ট্য:**
- সিমেট্রিক (Skewness = ০)
- গড় = মিডিয়ান = মোড
- এলাকার ৬৮% ±১ Std Dev এর মধ্যে
- এলাকার ৯৫% ±২ Std Dev এর মধ্যে
- এলাকার ৯৯.৭% ±৩ Std Dev এর মধ্যে

### স্ট্যান্ডার্ড নরমাল ডিস্ট্রিবিউশন
গড় = ০, Std Dev = ১

**Z-স্কোর:** Z = (X - μ) / σ

**উদাহরণ:**
> একটি স্টকের গড় রিটার্ন ১২%, Std Dev ১৫%। রিটার্ন ২০% হওয়ার Z-স্কোর কত?
>
> Z = (২০ - ১২) / ১৫ = ৮/১৫ = ০.৫৩৩
>
> Z-টেবিল থেকে Z=০.৫৩৩ এর জন্য CDF = ০.৭০৩
> মানে ৭০.৩% সময় রিটার্ন ২০% এর নিচে থাকবে।

### নরমাল ডিস্ট্রিবিউশনের নিয়ম (Empirical Rule)

| Z-স্কোর | এলাকা | ব্যাখ্যা |
|---------|-------|----------|
| ±১ | ৬৮% | ৬৮% ডেটা গড়ের ±১σ এর মধ্যে |
| ±২ | ৯৫% | ৯৫% ডেটা গড়ের ±২σ এর মধ্যে |
| ±৩ | ৯৯.৭% | প্রায় সব ডেটা ±৩σ এর মধ্যে |

## 📖 5. Lognormal Distribution

স্টকের দামের জন্য ব্যবহৃত — কারণ দাম কখনো ঋণাত্মক হতে পারে না।

**বৈশিষ্ট্য:**
- শুধুমাত্র পজিটিভ মান
- Right-Skewed (পজিটিভ স্কিউ)
- ln(Price) Normal Distribution অনুসরণ করে

**ফাইন্যান্সে ব্যবহার:**
> স্টকের দাম Lognormal হয় (০ থেকে অসীম)।
> স্টকের রিটার্ন (ln(P₂/P₁)) Normal হয়।

## 📖 6. Poisson Distribution

নির্দিষ্ট সময়ে বিরল ঘটনা গণনা করে।

**সূত্র:** P(X=k) = (λ^k × e^(-λ)) / k!

**যেখানে λ = গড় ঘটনার সংখ্যা**

**উদাহরণ:**
> একটি স্টক এক্সচেঞ্জে প্রতিদিন গড়ে ২টি বড় প্রাইস মুভমেন্ট হয়।
> একটি দিনে ঠিক ৩টি বড় মুভমেন্ট হওয়ার সম্ভাবনা:
>
> P(X=৩) = (২³ × e⁻²) / ৩!
> = (৮ × ০.১৩৫৩) / ৬
> = **১৮%**

## ✍️ অনুশীলন (Practice Questions)

### প্রশ্ন ১:
একটি স্টকের গড় রিটার্ন ১০%, Std Dev ২০%। রিটার্ন ৩০% এর কম হওয়ার সম্ভাবনা কত (Normal Distribution ধরে)?

**সমাধান:**
Z = (৩০ - ১০) / ২০ = ১.০
Z=১.০ এর জন্য CDF = ০.৮৪১৩
**সম্ভাবনা = ৮৪.১৩%**

### প্রশ্ন ২:
একজন বিনিয়োগকারী ৬০% সময় সঠিক সিদ্ধান্ত নেন। তিনি ৮টি সিদ্ধান্ত নিলে ঠিক ৫টিতে সঠিক হওয়ার সম্ভাবনা কত?

**সমাধান:**
P = C(৮,৫) × (০.৬)⁵ × (০.৪)³
= ৫৬ × ০.০৭৭৭৬ × ০.০৬৪
= ৫৬ × ০.০০৪৯৮
= **২৭.৮৮%**

### প্রশ্ন ৩:
একটি পোর্টফোলিওর রিটার্ন Normally distributed, গড় ১৫%, Std Dev ১০%।
ক) ১৫% থেকে ২৫% রিটার্নের সম্ভাবনা কত?
খ) ৫% এর কম রিটার্নের সম্ভাবনা কত?

**সমাধান:**
ক) Z₁ = ০, Z₂ = (২৫-১৫)/১০ = ১.০
    P(Z<১) = ০.৮৪১৩, P(Z<০) = ০.৫০০০
    **P = ০.৮৪১৩ - ০.৫০০০ = ৩৪.১৩%**

খ) Z = (৫-১৫)/১০ = -১.০
    **P = ০.১৫৮৭ = ১৫.৮৭%**

## 📝 সূত্র সংক্ষেপ

| ধারণা | সূত্র |
|-------|-------|
| প্রত্যাশিত মান | E(X) = Σx·P(x) |
| ভ্যারিয়েন্স | Var(X) = Σ(x-μ)²·P(x) |
| Z-স্কোর | Z = (X - μ) / σ |
| Binomial | C(n,k)·p^k·(1-p)^(n-k) |
| Poisson | λ^k·e^(-λ) / k! |

## 🎯 ডেটা অ্যানালিস্ট টিপস

**Python দিয়ে Distribution বিশ্লেষণ:**
```python
import numpy as np
from scipy import stats

# Normal Distribution
mean, std = 0.12, 0.15
prob = stats.norm.cdf(0.20, mean, std)
print(f"P(return < 20%): {prob:.2%}")

# Z-score থেকে সম্ভাবনা
z = (0.20 - 0.12) / 0.15
prob_z = stats.norm.cdf(z)
print(f"P(Z < {z:.2f}): {prob_z:.2%}")

# Binomial Distribution
n, p, k = 8, 0.6, 5
prob_binom = stats.binom.pmf(k, n, p)
print(f"P(X={k}) for B({n},{p}): {prob_binom:.2%}")

# রিটার্নের মন্টি কার্লো সিমুলেশন
sim_returns = np.random.normal(0.12, 0.15, 10000)
var_95 = np.percentile(sim_returns, 5)
print(f"Value at Risk (95%): {var_95:.2%}")
```
