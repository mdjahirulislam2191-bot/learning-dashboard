## 📝 Part 1: TVM (Day 8)

### প্রশ্ন ১:
আপনি $১০,০০০ ৮% বার্ষিক চক্রবৃদ্ধি হারে ৫ বছরের জন্য বিনিয়োগ করলেন। ভবিষ্যৎ মূল্য (FV) কত?

A) $১৪,০০০
B) $১৪,৬৯৩.২৮
C) $১৫,০০০
D) $১৩,৬০০

<details>
<summary>সমাধান</summary>
FV = PV × (1+r)ⁿ = $১০,০০০ × (১.০৮)⁵
FV = $১০,০০০ × ১.৪৬৯৩৩
**উত্তর: B — $১৪,৬৯৩.২৮**
</details>

### প্রশ্ন ২:
আপনি ১০ বছর পর $৫০,০০০ পেতে চান। বার্ষিক চক্রবৃদ্ধি ১২% হলে আজকে কত বিনিয়োগ করতে হবে?

A) $১৫,০০০
B) $১৬,০৯৮.৬২
C) $২০,০০০
D) $২৫,০০০

<details>
<summary>সমাধান</summary>
PV = $৫০,০০০ / (১.১২)¹⁰
PV = $৫০,০০০ / ৩.১০৫৮৫
**উত্তর: B — $১৬,০৯৮.৬২**
</details>

### প্রশ্ন ৩:
আপনি ৩ বছর ধরে প্রতি মাসের শেষে $২০০ জমা দিচ্ছেন। মাসিক সুদের হার ০.৫%। ভবিষ্যৎ মূল্য কত?

A) $৭,২০০
B) $৭,৮২৩.৪৬
C) $৮,০০০
D) $৭,৫০০

<details>
<summary>সমাধান</summary>
n = ৩৬, r = ০.০০৫
FVA = $২০০ × [(১.০০৫)³⁶ - ১] / ০.০০৫
FVA = $২০০ × ৩৯.৩৩৬১
**উত্তর: B — $৭,৮২৩.৪৬**
</details>

## 📝 Part 2: DCF (Day 9)

### প্রশ্ন ৪:
একটি প্রকল্পে প্রাথমিক বিনিয়োগ $২০,০০০। আগামী ৪ বছরে ক্যাশ ফ্লো: $৬,০০০, $৭,০০০, $৮,০০০, $৯,০০০। ডিসকাউন্ট রেট ১০%। NPV কত?

A) $২,০০০
B) $২,৮৫২.৬৫
C) $৩,০০০
D) $৪,০০০

<details>
<summary>সমাধান</summary>
PV₁ = $৬,০০০ / ১.১০ = $৫,৪৫৪.৫৫
PV₂ = $৭,০০০ / (১.১০)² = $৫,৭৮৫.১২
PV₃ = $৮,০০০ / (১.১০)³ = $৬,০১০.৫২
PV₄ = $৯,০০০ / (১.১০)⁴ = $৬,১৪৭.১২

মোট PV = $২৩,৩৯৭.৩১
NPV = $২৩,৩৯৭.৩১ - $২০,০০০
**উত্তর: B — $২,৩৯৭.৩১** (দ্বিতীয় অপশনের কাছাকাছি)
*দ্রষ্টব্য: প্রকৃত NPV $২,৩৯৭.৩১*
</details>

### প্রশ্ন ৫:
একটি স্টক প্রতি বছর $২ ডিভিডেন্ড দেয় যা ৪% হারে বৃদ্ধি পায়। প্রয়োজনীয় রিটার্ন ১০%। স্টকের মূল্য কত?

A) $২০
B) $২৫
C) $৩৩.৩৩
D) $৫০

<details>
<summary>সমাধান</summary>
D₁ = $২
P = D₁ / (r - g) = $২ / (০.১০ - ০.০৪)
P = $২ / ০.০৬
**উত্তর: C — $৩৩.৩৩**
</details>

### প্রশ্ন ৬:
একটি পারপিচুইটি প্রতি বছর $১,০০০ দেয়। ডিসকাউন্ট রেট ৫%। বর্তমান মূল্য কত?

A) $৫,০০০
B) $১০,০০০
C) $১৫,০০০
D) $২০,০০০

<details>
<summary>সমাধান</summary>
PV = PMT / r = $১,০০০ / ০.০৫
**উত্তর: D — $২০,০০০**
</details>

## 📝 Part 3: Statistics (Day 10)

### প্রশ্ন ৭:
একটি পোর্টফোলিওর ৪ বছরের রিটার্ন: ৮%, ১৫%, -৫%, ১২%। গাণিতিক গড় কত?

B) ৭.০%

<details>

**উত্তর: C — ৭.৫%**

## 📝 Part 4: Distributions (Day 11)

### প্রশ্ন ১০:
একটি স্টকের রিটার্ন Normally distributed: গড় ১৬%, Std Dev ২৪%। রিটার্ন ২৮% এ Z-স্কোর কত?

A) ০.২৫
B) ০.৫০
C) ০.৭৫
D) ১.০০

<details>
<summary>সমাধান</summary>
Z = (২৮ - ১৬) / ২৪ = ১২/২৪
**উত্তর: B — ০.৫০**
</details>

### প্রশ্ন ১১:
কোন বিতরণ ঋণাত্মক মান নিতে পারে না এবং ডান দিকে skewed?

A) Normal
B) Binomial
C) Lognormal
D) Poisson

<details>
<summary>সমাধান</summary>
Lognormal Distribution শুধুমাত্র পজিটিভ মান নেয় এবং Right-Skewed।
**উত্তর: C — Lognormal**
</details>

### প্রশ্ন ১২:
Normal Distribution অনুযায়ী, ডেটার কত শতাংশ গড়ের ±২ Std Dev এর মধ্যে থাকে?

A) ৬৮%
B) ৯৫%
C) ৯৯.৭%
D) ৫০%

<details>
<summary>সমাধান</summary>
Empirical Rule অনুযায়ী: ±২σ = ৯৫%
**উত্তর: B — ৯৫%**
</details>

## 📝 Part 5: Sampling (Day 12)

### প্রশ্ন ১৩:
একটি পোর্টফোলিওর ৪৯টি মাসিক রিটার্নের নমুনায় Std Dev ২১%। Standard Error কত?

A) ২%
B) ৩%
C) ৪%
D) ৫%

<details>
<summary>সমাধান</summary>
SE = σ / √n = ২১% / √৪৯ = ২১% / ৭
**উত্তর: B — ৩%**
</details>

### প্রশ্ন ১৪:
৯৫% Confidence Level এ, ১০০টি স্টকের গড় রিটার্ন ১১% এবং Std Dev ১৫% হলে Confidence Interval কত?

A) [৮%, ১৪%]
B) [৮.০৬%, ১৩.৯৪%]
C) [১০%, ১২%]
D) [৯.৫%, ১২.৫%]

<details>
<summary>সমাধান</summary>
SE = ১৫% / √১০০ = ১.৫%
CI = ১১% ± (১.৯৬ × ১.৫%)
CI = ১১% ± ২.৯৪%
**উত্তর: B — [৮.০৬%, ১৩.৯৪%]**
</details>

### প্রশ্ন ১৫:
কেন স্যাম্পলিং ব্যবহার করা হয়? সবচেয়ে উপযুক্ত কারণ কী?

A) নমুনা সবসময় নির্ভুল
B) সময় ও খরচ বাঁচায়
C) নমুনা জনসংখ্যার চেয়ে বেশি ডেটা দেয়
D) নমুনার Std Dev বেশি থাকে

<details>
<summary>সমাধান</summary>
সম্পূর্ণ জনসংখ্যা পরীক্ষা করা সময়সাপেক্ষ ও ব্যয়বহুল, তাই নমুনা ব্যবহার করা হয়।

</details>
