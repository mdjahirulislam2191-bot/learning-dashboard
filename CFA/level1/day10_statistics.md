## 📖 1. পরিসংখ্যানের প্রকারভেদ

### বর্ণনামূলক পরিসংখ্যান (Descriptive Statistics)
ডেটা সংক্ষিপ্ত ও বর্ণনা করে।

**উদাহরণ:** একটি স্টকের গড় রিটার্ন ১২%, স্ট্যান্ডার্ড ডেভিয়েশন ১৫%।

### অনুমানমূলক পরিসংখ্যান (Inferential Statistics)
নমুনা থেকে জনসংখ্যা সম্পর্কে অনুমান।

**উদাহরণ:** ৫০টি স্টকের উপর ভিত্তি করে BSE ৫০০-এর গড় রিটার্ন অনুমান করা।

## 📖 2. কেন্দ্রীয় প্রবণতা (Central Tendency)

### গাণিতিক গড় (Arithmetic Mean)
`μ = ΣX / N` (জনসংখ্যা)
`x̄ = Σx / n` (নমুনা)

**উদাহরণ:** স্টক রিটার্ন: ১০%, ১৫%, ৮%, ১২%, ২০%
গড় = (১০+১৫+৮+১২+২০)/৫ = ১৩%

### জ্যামিতিক গড় (Geometric Mean)
চক্রবৃদ্ধি রিটার্নের জন্য ব্যবহার করা হয়।

`G = [(1+R₁)(1+R₂)...(1+Rₙ)]^(1/n) - 1`

**উদাহরণ:**
> বিনিয়োগ রিটার্ন: +২০%, -১৫%, +২৫%
> GM = [(১.২০)(০.৮৫)(১.২৫)]^(১/৩) - ১
> GM = (১.২৭৫)^(১/৩) - ১
> GM = ১.০৮৪ - ১ = **৮.৪%**

> গাণিতিক গড় = (২০ - ১৫ + ২৫)/৩ = **১০%**
>
> লক্ষ্য করুন: GM < AM (যখন ডেটায় ভ্যারিয়েশন থাকে)

### মিডিয়ান (Median)
মাঝখানের মান — ডেটা সাজিয়ে।

**উদাহরণ:** {৮, ১০, ১২, ১৫, ২০} → মিডিয়ান = ১২

### মোড (Mode)
সবচেয়ে বেশি বার আসা মান।

**উদাহরণ:** {৫, ৭, ৭, ৯, ১১, ১৩, ১৩, ১৩, ১৫} → মোড = ১৩

## 📖 3. কোয়ান্টাইল (Quantiles)

ডেটাকে সমান ভাগে ভাগ করা।

| প্রকার | ভাগ | বিবরণ |
|--------|-----|-------|
| Quartile | ৪ ভাগ | Q1, Q2 (Median), Q3 |
| Quintile | ৫ ভাগ | পাঁচ ভাগে ভাগ |
| Decile | ১০ ভাগ | দশ ভাগে ভাগ |
| Percentile | ১০০ ভাগ | শতাংশ ভিত্তিক |

**উদাহরণ — ফাইন্যান্সে ব্যবহার:**
> একটি মিউচুয়াল ফান্ড রিটার্নের ৯০ তম Percentile-এ আছে। অর্থাৎ, ৯০% ফান্ডের চেয়ে ভালো পারফর্ম করেছে।

## 📖 4. বিচ্ছুরণ (Dispersion)

### রেঞ্জ (Range)
সর্বোচ্চ - সর্বনিম্ন

**উদাহরণ:** {৮, ১০, ১২, ১৫, ২০} → রেঞ্জ = ২০ - ৮ = ১২

### ভ্যারিয়েন্স (Variance)
গড় থেকে বর্গিত বিচ্যুতির গড়।

**জনসংখ্যা ভ্যারিয়েন্স:** `σ² = Σ(X - μ)² / N`

**নমুনা ভ্যারিয়েন্স:** `s² = Σ(x - x̄)² / (n-1)`

### স্ট্যান্ডার্ড ডেভিয়েশন (Standard Deviation)
ভ্যারিয়েন্সের বর্গমূল।

`σ = √σ²` বা `s = √s²`

**উদাহরণ — স্ট্যান্ডার্ড ডেভিয়েশন গণনা:**
> স্টক রিটার্ন: ১০%, ১২%, ১৪%, ১৬%, ১৮%
> গড় = ১৪%
>
> বিচ্যুতি: (-৪)² + (-২)² + (০)² + (২)² + (৪)² = ৪০
> ভ্যারিয়েন্স = ৪০/৫ = ৮ (জনসংখ্যা)
> Std Dev = √৮ = **২.৮৩%**

### অর্থে Std Dev এর ব্যাখ্যা:

| Std Dev | ঝুঁকি স্তর | উদাহরণ |
|---------|------------|--------|
| < ১০% | কম | ট্রেজারি বন্ড |
| ১০-২০% | মাঝারি | ব্লু-চিপ স্টক |
| > ২০% | বেশি | ছোট ক্যাপ/ইমার্জিং স্টক |

## 📖 5. ডেটার আকৃতি (Shape of Distribution)

### স্কিউনেস (Skewness)
ডেটার অসমতা পরিমাপ করে।

- **পজিটিভ স্কিউ (Right-Skewed):** গড় > মিডিয়ান > মোড
  - বেশি ছোট মান, কম বড় মান
  - উদাহরণ: ছোট স্টকের রিটার্ন

- **নেগেটিভ স্কিউ (Left-Skewed):** গড় < মিডিয়ান < মোড
  - কম ছোট মান, বেশি বড় মান
  - উদাহরণ: বন্ড রিটার্ন

- **সিমেট্রিক (Normal):** গড় = মিডিয়ান = মোড

### কার্টোসিস (Kurtosis)
ডেটার চূড়া (Peak) ও লেজ (Tail) এর পুরুত্ব।

- **Leptokurtic:** মোটা লেজ → বেশি চরম ঘটনা
- **Mesokurtic:** সাধারণ (Normal)
- **Platykurtic:** পাতলা লেজ → কম চরম ঘটনা

**ফাইন্যান্সে গুরুত্ব:**
> স্টক রিটার্ন সাধারণত Leptokurtic হয় — মানে বড় লোকসানের সম্ভাবনা Normal Distribution-এর চেয়ে বেশি।

## ✍️ অনুশীলন (Practice Questions)

### প্রশ্ন ১:
একটি পোর্টফোলিওর ৫ বছরের রিটার্ন: ৮%, -২%, ১৪%, ৬%, ২২%। গাণিতিক গড় ও জ্যামিতিক গড় বের করুন।

**সমাধান:**
গাণিতিক গড় = (৮-২+১৪+৬+২২)/৫ = ৪৮/৫ = **৯.৬%**
জ্যামিতিক গড় = [(১.০৮)(০.৯৮)(১.১৪)(১.০৬)(১.২২)]^(১/৫) - ১
= (১.৫১৯৭)^(০.২) - ১ = ১.০৮৭ - ১ = **৮.৭%**

### প্রশ্ন ২:
ডেটাসেট {১২, ১৫, ১৮, ২০, ২৫, ২৮, ৩০} এর মিডিয়ান ও কোয়ার্টাইল বের করুন।

**সমাধান:**
মিডিয়ান (Q2) = ২০
Q1 = (১৫+১৮)/২ = ১৬.৫
Q3 = (২৫+২৮)/২ = ২৬.৫

### প্রশ্ন ৩:
স্টক A: গড় রিটার্ন ১২%, Std Dev ১৫%
স্টক B: গড় রিটার্ন ১২%, Std Dev ২৫%
কোন স্টক বেশি ঝুঁকিপূর্ণ?

**উত্তর:** স্টক B বেশি ঝুঁকিপূর্ণ (উচ্চ Std Dev), যদিও গড় রিটার্ন একই।

## 📝 সূত্র সংক্ষেপ

| ধারণা | সূত্র |
|-------|-------|
| গাণিতিক গড় | ΣX / n |
| জ্যামিতিক গড় | [(1+R₁)...(1+Rₙ)]^(1/n) - 1 |
| ভ্যারিয়েন্স (জনসংখ্যা) | Σ(X - μ)² / N |
| ভ্যারিয়েন্স (নমুনা) | Σ(x - x̄)² / (n-1) |
| Std Dev | √Variance |
| Coeff of Variation | Std Dev / Mean |

## 🎯 ডেটা অ্যানালিস্ট টিপস

**Python দিয়ে পরিসংখ্যান:**
```python
import numpy as np
import scipy.stats as stats

returns = [0.10, -0.02, 0.14, 0.06, 0.22]

arithmetic_mean = np.mean(returns)
geometric_mean = np.prod([1+r for r in returns])**(1/5) - 1
variance = np.var(returns, ddof=0)  # population
std_dev = np.std(returns, ddof=0)
skewness = stats.skew(returns)
kurtosis = stats.kurtosis(returns)

print(f"Mean: {arithmetic_mean:.2%}")
print(f"Geometric Mean: {geometric_mean:.2%}")
print(f"Std Dev: {std_dev:.2%}")
print(f"Skewness: {skewness:.2f}")
print(f"Excess Kurtosis: {kurtosis:.2f}")
```
