# 📘 Day 34: Equity — Dividend Discount Model (DDM)

## 🎯 শেখার উদ্দেশ্য

এই পাঠ শেষে আপনি যা পারবেন:
1. ডিভিডেন্ড ডিসকাউন্ট মডেল (DDM) বুঝতে
2. গর্ডন গ্রোথ মডেল (GGM) বুঝতে
3. মাল্টি-স্টেজ DDM বুঝতে
4. DDM-এর সীমাবদ্ধতা বুঝতে

## 📖 1. DDM-এর মূল ধারণা

**DDM বলে:** একটি স্টকের মূল্য = তার ভবিষ্যতের সকল ডিভিডেন্ডের বর্তমান মূল্যের সমষ্টি।

**সাধারণ সূত্র:** `V₀ = Σ Dt / (1+r)ᵗ`

**যেখানে:**
- V₀ = বর্তমান মূল্য
- Dt = t সময়ে ডিভিডেন্ড
- r = প্রয়োজনীয় রিটার্ন (Cost of Equity)

---

## 📖 2. গর্ডন গ্রোথ মডেল (GGM / Constant Growth DDM)

**অনুমান:** ডিভিডেন্ড ধ্রুবক হারে (g) বৃদ্ধি পাবে।

**সূত্র:** `V₀ = D₁ / (r - g)`

**যেখানে:**
- D₁ = পরবর্তী বছরের ডিভিডেন্ড = D₀ × (1+g)
- r = প্রয়োজনীয় রিটার্ন
- g = ডিভিডেন্ড বৃদ্ধির হার

**উদাহরণ:**
> D₀ = $২.০০, g = ৫%, r = ১১%
> D₁ = $২.০০ × ১.০৫ = $২.১০
> V₀ = $২.১০ / (০.১১ - ০.০৫) = $২.১০ / ০.০৬ = **$৩৫.০০**

### GGM-এর শর্ত
- r > g (নতুবা মান অসীম হবে)
- কোম্পানি ডিভিডেন্ড দেয়
- বৃদ্ধির হার স্থিতিশীল

## 📖 3. মাল্টি-স্টেজ DDM

যখন ডিভিডেন্ডের বৃদ্ধি হার পরিবর্তন হয় — সাধারণত টু-স্টেজ বা থ্রি-স্টেজ মডেল।

**টু-স্টেজ মডেল:**
```
V₀ = Σ [D₀(1+g₁)ᵗ/(1+r)ᵗ] + Vn/(1+r)ⁿ
```
যেখানে Vn = Dn₊₁/(r-g₂)

**উদাহরণ:**
> কোম্পানি XYZ:
> প্রথম ৩ বছর: g₁ = ১৫% (উচ্চ বৃদ্ধি)
> তারপর: g₂ = ৫% (স্থিতিশীল বৃদ্ধি)
> r = ১২%, D₀ = $১.০০
>
> ধাপ ১: প্রথম ৩ বছরের PV
> D₁ = $১.১৫, PV = $১.১৫/১.১২ = $১.০২৭
> D₂ = $১.৩২২৫, PV = $১.০৫৪
> D₃ = $১.৫২০৯, PV = $১.০৮২
>
> ধাপ ২: টার্মিনাল ভ্যালু
> D₄ = $১.৫২০৯ × ১.০৫ = $১.৫৯৬৯
> V₃ = $১.৫৯৬৯/(০.১২-০.০৫) = $২২.৮১৩

## 📖 4. গর্ডন গ্রোথ মডেল থেকে রিটার্ন (Implied Return)

`r = D₁/P₀ + g`

**উদাহরণ:**
> P₀ = $৪০, D₁ = $২.৪০, g = ৪%
> r = $২.৪০/$৪০ + ৪% = ৬% + ৪% = **১০%**
