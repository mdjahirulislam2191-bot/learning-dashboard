# 📘 Day 9: Quant — Discounted Cash Flow (DCF)

## 🎯 শেখার উদ্দেশ্য

এই পাঠ শেষে আপনি যা পারবেন:
1. DCF পদ্ধতি ব্যাখ্যা করতে
2. নেট প্রেজেন্ট ভ্যালু (NPV) গণনা করতে
3. ইন্টারনাল রেট অফ রিটার্ন (IRR) বের করতে
4. বিভিন্ন ধরনের ক্যাশ ফ্লো অ্যানালাইসিস করতে
5. পোর্টফোলিও রিটার্ন গণনা করতে

---

## 📖 1. DCF এর মূল ধারণা

DCF হলো ভবিষ্যতের নগদ প্রবাহকে বর্তমান মূল্যে নিয়ে আসার একটি পদ্ধতি।

**মূল ধারণা:** একটি বিনিয়োগের মূল্য = তার ভবিষ্যতের সকল নগদ প্রবাহের বর্তমান মূল্যের সমষ্টি।

**সূত্র:** `NPV = Σ [CFt / (1+r)ᵗ] - Initial Investment`

**যেখানে:**
- CFt = t সময়ে নগদ প্রবাহ
- r = ডিসকাউন্ট রেট
- t = সময়কাল

---

## 📖 2. নেট প্রেজেন্ট ভ্যালু (NPV)

NPV হলো বিনিয়োগের ভবিষ্যতের ক্যাশ ফ্লোর বর্তমান মূল্য এবং প্রাথমিক বিনিয়োগের পার্থক্য।

**NPV নিয়ম:**
- NPV > 0: বিনিয়োগ লাভজনক ✓
- NPV < 0: বিনিয়োগ লাভজনক নয় ✗
- NPV = 0: ব্রেক-ইভেন

**উদাহরণ — NPV গণনা:**
> একটি প্রকল্পে প্রাথমিক বিনিয়োগ $১০,০০০
> আগামী ৩ বছরে ক্যাশ ফ্লো: $৪,০০০, $৪,৫০০, $৫,০০০
> ডিসকাউন্ট রেট: ১০%
>
> PV₁ = $৪,০০০ / (১.১০)¹ = $৩,৬৩৬.৩৬
> PV₂ = $৪,৫০০ / (১.১০)² = $৩,৭১৯.০১
> PV₃ = $৫,০০০ / (১.১০)³ = $৩,৭৫৬.৫৭
>
> মোট PV = $১১,১১১.৯৪
> NPV = $১১,১১১.৯৪ - $১০,০০০ = **$১,১১১.৯৪**
>
> NPV > 0 হওয়ায় প্রকল্পটি গ্রহণযোগ্য ✅

---

## 📖 3. ইন্টারনাল রেট অফ রিটার্ন (IRR)

IRR হলো সেই ডিসকাউন্ট রেট যেখানে NPV = ০ হয়।

IRR নিয়ম:
- IRR > Cost of Capital: বিনিয়োগ গ্রহণযোগ্য ✓
- IRR < Cost of Capital: বিনিয়োগ গ্রহণযোগ্য নয় ✗

**NPV Profile:**
```
NPV |
    |
+ve |\        (IRR = ক্রসিং পয়েন্ট)
    | \
  0 |--\---------> Discount Rate
    |   \
-ve |    \
```

**উদাহরণ — IRR গণনা (আনুমানিক):**
> প্রাথমিক বিনিয়োগ: $১,০০০
> ক্যাশ ফ্লো: বছরে $৪০০, ৩ বছর
>
> ১০% এ NPV: $৪০০ × [১-(১.১০)⁻³]/০.১০ - $১,০০০ = -$৫.২৬
> ৯% এ NPV: $৪০০ × [১-(১.০৯)⁻³]/০.০৯ - $১,০০০ = $১২.৫২
>
> IRR ≈ ৯.৭% (১০% ও ৯% এর মধ্যে Interpolate করে)

## 📖 4. ক্যাশ ফ্লোর প্রকারভেদ

### একক (Single) ক্যাশ ফ্লো
ভবিষ্যতে একবারে একটি পেমেন্ট।

### অ্যানুইটি (Annuity)
নিয়মিত সমান পেমেন্ট।

### প erপিচুইটি (Perpetuity)
অসীম সময় ধরে সমান পেমেন্ট।

**পারপিচুইটির PV:** `PV = PMT / r`

**উদাহরণ — পারপিচুইটি:**
> একটি বন্ড প্রতি বছর $১০০ দেয়, অসীম সময় ধরে। ডিসকাউন্ট রেট ৫%।
> PV = $১০০ / ০.০৫ = $২,০০০

### গ্রোয়িং পারপিচুইটি (Growing Perpetuity)
অসীম সময় ধরে নির্দিষ্ট হারে বাড়তে থাকা পেমেন্ট।

**সূত্র:** `PV = PMT / (r - g)`

**যেখানে g = বৃদ্ধির হার**

**উদাহরণ — গ্রোয়িং পারপিচুইটি:**
> প্রথম বছর $১০০, প্রতি বছর ২% হারে বৃদ্ধি পাচ্ছে। ডিসকাউন্ট রেট ৮%।
> PV = $১০০ / (০.০৮ - ০.০২) = $১০০ / ০.০৬ = $১,৬৬৬.৬৭

## 📖 5. পোর্টফোলিও রিটার্ন গণনা

### গড় রিটার্ন (Mean Return)
`R_avg = (R₁ + R₂ + ... + Rₙ) / n`

### সময়-ভারিত রিটার্ন (Time-Weighted Return)
`TWR = [(1+R₁)(1+R₂)...(1+Rₙ)]^(1/n) - 1`

### অর্থ-ভারিত রিটার্ন (Money-Weighted Return)
প্রকৃতপক্ষে এটি IRR — ক্যাশ ফ্লোর উপর ভিত্তি করে।

**ডেটা অ্যানালিস্ট উদাহরণ:**
> একটি পোর্টফোলিওর মাসিক রিটার্ন: +২%, -১%, +৩%, +১.৫%
> গাণিতিক গড়: (২ - ১ + ৩ + ১.৫) / ৪ = ১.৩৭৫%
> সময়-ভারিত রিটার্ন: [(১.০২)(০.৯৯)(১.০৩)(১.০১৫)]^(১/৪) - ১ = ১.৩৪%

## ✍️ অনুশীলন (Practice Questions)

### প্রশ্ন ১:
একটি প্রকল্পে প্রাথমিক বিনিয়োগ $৫০,০০০ এবং ৫ বছর ধরে প্রতি বছর $১৫,০০০ ক্যাশ ফ্লো। ডিসকাউন্ট রেট ১২%। NPV কত?

**সমাধান:**
PV = $১৫,০০০ × [১-(১.১২)⁻⁵] / ০.১২
PV = $১৫,০০০ × ৩.৬০৪৮
PV = $৫৪,০৭২
NPV = $৫৪,০৭২ - $৫০,০০০ = **$৪,০৭২ ✅**

### প্রশ্ন ২:
একটি কোম্পানি $২০,০০০ বিনিয়োগ করবে যা ৪ বছর ধরে $৬,০০০ করে দেবে। IRR আনুমানিক কত?

**সমাধান:**
১০%: NPV = $৬,০০০ × ৩.১৬৯৯ - $২০,০০০ = -$৯৮০.৬০
৮%: NPV = $৬,০০০ × ৩.৩১২১ - $২০,০০০ = -$১২৭.৪০
৭%: NPV = $৬,০০০ × ৩.৩৮৭২ - $২০,০০০ = $৩২৩.২০
**IRR ≈ ৭.৭%**

### প্রশ্ন ৩:
একটি স্টক প্রতি বছর $৫ ডিভিডেন্ড দেয় এবং প্রতি বছর ৩% হারে বৃদ্ধি পায়। প্রয়োজনীয় রিটার্ন ১১%। স্টকের মূল্য কত?

**সমাধান:**
P = D₁ / (r - g) = $৫ / (০.১১ - ০.০৩) = $৫ / ০.০৮
**P = $৬২.৫০**

## 📝 সূত্র সংক্ষেপ

| ধারণা | সূত্র |
|-------|-------|
| NPV | Σ CFt/(1+r)ᵗ - Initial Investment |
| IRR | Σ CFt/(1+IRR)ᵗ = Initial Investment |
| পারপিচুইটি PV | PV = PMT / r |
| গ্রোয়িং পারপিচুইটি PV | PV = PMT / (r - g) |
| TWR | [(1+R₁)...(1+Rₙ)]^(1/n) - 1 |

## 🎯 ডেটা অ্যানালিস্ট টিপস

**Python দিয়ে DCF বিশ্লেষণ:**
```python
# NPV ফাংশন
def npv(cash_flows, rate):
    return sum(cf / (1 + rate) ** t for t, cf in enumerate(cash_flows))

# IRR ফাংশন (Newton Method)
def irr(cash_flows, guess=0.1):
    rate = guess
    for _ in range(1000):
        npv_val = npv(cash_flows, rate)
        dnpv = sum(-t * cf / (1 + rate) ** (t+1) 
                   for t, cf in enumerate(cash_flows))
        rate -= npv_val / dnpv
    return rate

# উদাহরণ
cfs = [-50000, 15000, 15000, 15000, 15000, 15000]
print(f"NPV (12%): ${npv(cfs, 0.12):.2f}")
print(f"IRR: {irr(cfs):.2%}")
```
