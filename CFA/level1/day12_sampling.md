এই পাঠ শেষে আপনি যা পারবেন:

## 📖 1. জনসংখ্যা ও নমুনা

**জনসংখ্যা (Population):** সম্পূর্ণ ডেটা সেট
**নমুনা (Sample):** জনসংখ্যার একটি অংশ

**কেন নমুনা ব্যবহার করি?**
- সম্পূর্ণ জনসংখ্যা পরীক্ষা করা সময়সাপেক্ষ ও ব্যয়বহুল
- কিছু পরীক্ষা ধ্বংসাত্মক
- নমুনা দিয়েই ভালো অনুমান করা যায়

**উদাহরণ (Data Analyst context):**
> আপনি কানাডার সকল ব্যাংকের ঋণখেলাপির হার বের করতে চান। ১০০টি ব্যাংকের ডেটা নেওয়া সহজ নয়। ২০টি ব্যাংকের নমুনা নিয়ে বিশ্লেষণ করলেই ভালো অনুমান পাওয়া যায়।

---

## 📖 2. নমুনা পদ্ধতি (Sampling Methods)

### সম্ভাবনা ভিত্তিক (Probability Sampling)

**সরল র্যান্ডম স্যাম্পলিং (Simple Random Sampling):**
প্রতিটি সদস্যের সমান সুযোগ।

**উদাহরণ:** S&P 500 থেকে ৫০টি কোম্পানি লটারি পদ্ধতিতে বেছে নেওয়া।

**স্তরিত নমুনা (Stratified Random Sampling):**
জনসংখ্যাকে স্তরে (Strata) ভাগ করে প্রতিটি স্তর থেকে নমুনা।

**উদাহরণ (Finance context):**
> মার্কেট ক্যাপ অনুযায়ী স্তরিত নমুনা:
> - Large Cap (>$১০B) থেকে ১০টি
> - Mid Cap ($২-১০B) থেকে ১০টি
> - Small Cap (<$২B) থেকে ১০টি

**ক্লাস্টার স্যাম্পলিং (Cluster Sampling):**
জনসংখ্যাকে ক্লাস্টারে ভাগ করে কিছু ক্লাস্টার বেছে নেওয়া।

### অ-সম্ভাবনা ভিত্তিক (Non-Probability Sampling)

**কনভিনিয়েন্স স্যাম্পলিং:** সহজলভ্য ডেটা ব্যবহার
**জাজমেন্টাল স্যাম্পলিং:** বিশেষজ্ঞের পছন্দ

## 📖 3. নমুনা বায়াস (Sample Bias)

### সিলেকশন বায়াস (Selection Bias)
নমুনা পুরো জনসংখ্যার প্রতিনিধিত্ব করে না।

**প্রকারভেদ:**
- **Survivorship Bias:** শুধুমাত্র টিকে থাকা কোম্পানি নিয়ে বিশ্লেষণ
- **Look-Ahead Bias:** ভবিষ্যতের তথ্য ব্যবহার করা
- **Time-Period Bias:** নির্দিষ্ট সময়ের ডেটা নেওয়া

**উদাহরণ — Survivorship Bias:**
> "মিউচুয়াল ফান্ডের গড় রিটার্ন ১২%" — যদি শুধুমাত্র এখনো বিদ্যমান ফান্ডের ডেটা নেওয়া হয়, তাহলে বাস্তবিক রিটার্ন কম হবে (ব্যর্থ ফান্ডগুলো বাদ পড়ে গেছে)।

### ডেটা-মাইনিং বায়াস (Data-Mining Bias)
একই ডেটায় বারবার টেস্ট করে প্যাটার্ন খোঁজা।

---

## 📖 4. কেন্দ্রীয় সীমা উপপাদ্য (Central Limit Theorem — CLT)

**CLT বলে:** পর্যাপ্ত বড় নমুনার (n ≥ 30) নমুনা গড়ের বিতরণ প্রায় Normal হয়, জনসংখ্যার বিতরণ যাই হোক না কেন।

**CLT এর বৈশিষ্ট্য:**
- নমুনা গড়ের গড় = জনসংখ্যার গড় (μ)
- নমুনা গড়ের Std Dev = σ/√n (একে Standard Error বলে)
- n বাড়লে Standard Error কমে

**ফাইন্যান্স উদাহরণ:**
> একটি স্টকের রিটার্নটি Normal না-ও হতে পারে। কিন্তু ৩০টি পর্যবেক্ষণের নমুনা গড় প্রায় Normal হবে। তাই আমরা Confidence Interval তৈরি করতে পারি।

---

## 📖 5. Standard Error

Standard Error = জনসংখ্যার Std Dev / √n

**উদাহরণ:**
> একটি পোর্টফোলিওর জনসংখ্যা Std Dev = ২০%। ১০০টি নমুনার Standard Error কত?
>
> SE = ২০% / √১০০ = ২০% / ১০ = **২%**

---

## 📖 6. আস্থা ব্যবধান (Confidence Interval)

**সূত্র:** CI = x̄ ± (Critical Value × SE)

**Z-মান (Confidence Level অনুযায়ী):**
| Confidence Level | Z-value |
|-----------------|---------|
| ৯০% | ১.৬৪৫ |
| ৯৫% | ১.৯৬০ |
| ৯৯% | ২.৫৭৬ |

**উদাহরণ — ৯৫% Confidence Interval:**
> ১০০টি স্টকের নমুনায় গড় রিটার্ন ১২%, Std Dev ১৮%।
>
> SE = ১৮% / √১০০ = ১.৮%
> CI = ১২% ± (১.৯৬ × ১.৮%)
> CI = ১২% ± ৩.৫৩%
> **CI = [৮.৪৭%, ১৫.৫৩%]**
>
> অর্থাৎ, ৯৫% আস্থা সহ বলা যায় জনসংখ্যার গড় রিটার্ন ৮.৪৭% থেকে ১৫.৫৩% এর মধ্যে।

---

## 📖 7. নমুনা আকার নির্ধারণ (Sample Size)

**সূত্র:** n = (Z × σ / ME)²

**যেখানে:**
- Z = Confidence Level এর Z-মান
- σ = জনসংখ্যার Std Dev
- ME = মার্জিন অফ এরর

**উদাহরণ:**
> ৯৫% Confidence Level এ, Std Dev ২০%, এবং ±৩% মার্জিন অফ এরর চাই।
>
> n = (১.৯৬ × ২০ / ৩)²
> n = (৩৯.২ / ৩)²
> n = (১৩.০৭)²
> **n = ১৭১টি (আনুমানিক)**

## ✍️ অনুশীলন (Practice Questions)

### প্রশ্ন ১:
একটি মিউচুয়াল ফান্ডের ৫০টি মাসিক রিটার্নের নমুনায় গড় ১.২% এবং Std Dev ৪%। ৯৫% Confidence Interval নির্ণয় করুন।

**সমাধান:**
SE = ৪% / √৫০ = ০.৫৬৬%
CI = ১.২% ± (১.৯৬ × ০.৫৬৬%)
CI = ১.২% ± ১.১১%
**CI = [০.০৯%, ২.৩১%]**

### প্রশ্ন ২:
কেন CLT গুরুত্বপূর্ণ?

**উত্তর:** CLT বলে যে নমুনা গড়ের বিতরণ প্রায় Normal হয় যখন n ≥ ৩০। এটি আমাদের Confidence Interval ও Hypothesis Testing করতে দেয়, এমনকি যখন জনসংখ্যার বিতরণ অজানা।

### প্রশ্ন ৩:
Survivorship Bias কেন একটি সমস্যা?

**উত্তর:** যদি আমরা শুধুমাত্র এখনো টিকে থাকা কোম্পানি/ফান্ড নিয়ে বিশ্লেষণ করি, তাহলে আমরা ব্যর্থ হওয়াগুলো বাদ দিই। ফলে গড় রিটার্ন বেশি দেখায় — এটি বিনিয়োগ সিদ্ধান্ত ভুল করতে পারে।

## 📝 সূত্র সংক্ষেপ

| ধারণা | সূত্র |
|-------|-------|
| Standard Error | σ / √n |
| Confidence Interval | x̄ ± Z × SE |
| নমুনা আকার | n = (Z × σ / ME)² |
| CLT | x̄ ~ N(μ, σ²/n) |

## 🎯 ডেটা অ্যানালিস্ট টিপস

**Python দিয়ে Sampling:**
```python
import numpy as np
from scipy import stats

# র্যান্ডম নমুনা
population = np.random.normal(0.12, 0.20, 10000)
sample = np.random.choice(population, size=100, replace=False)

# Confidence Interval
mean = np.mean(sample)
se = np.std(sample, ddof=0) / np.sqrt(100)
ci_low, ci_high = stats.norm.interval(0.95, loc=mean, scale=se)

print(f"Sample Mean: {mean:.2%}")
print(f"95% CI: [{ci_low:.2%}, {ci_high:.2%}]")

# CLT ডেমোনস্ট্রেশন
sample_means = []
for _ in range(1000):
    s = np.random.choice(population, size=30)
    sample_means.append(np.mean(s))

print(f"Mean of sample means: {np.mean(sample_means):.2%}")
print(f"Population mean: {np.mean(population):.2%}")
```
