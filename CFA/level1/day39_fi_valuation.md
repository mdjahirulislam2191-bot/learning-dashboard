# 📘 Day 39: Fixed Income — Valuation

## 🎯 শেখার উদ্দেশ্য

এই পাঠ শেষে আপনি যা পারবেন:
1. বন্ড মূল্যায়নের মৌলিক নীতি বুঝতে
2. বন্ডের দাম ও YTM-এর সম্পর্ক বুঝতে
3. বন্ড প্রাইসিং সূত্র বুঝতে
4. প্রিমিয়াম বনাম ডিসকাউন্ট বন্ড বুঝতে
5. Accrued Interest বুঝতে

---

## 📖 1. বন্ড মূল্যায়নের নীতি

**মূলনীতি:** বন্ডের দাম = ভবিষ্যতের কুপন পেমেন্ট + Par Value-এর বর্তমান মূল্য।

**সূত্র:**
```
বন্ড মূল্য = Σ [C / (1+YTM)ᵗ] + [Par / (1+YTM)ⁿ]
```

**যেখানে:**
- C = কুপন পেমেন্ট (Coupon Rate × Par)
- YTM = Yield to Maturity (ডিসকাউন্ট রেট)
- n = মেয়াদ (বছর)
- Par = $১,০০০ (সাধারণত)

---

## 📖 2. বন্ডের দাম ও YTM-এর সম্পর্ক

বিপরীত সম্পর্ক: **YTM বাড়লে বন্ডের দাম কমে, YTM কমলে বাড়ে।**

### বন্ডের অবস্থা

| অবস্থা | শর্ত |
|--------|------|
| **Par Bond** | Coupon Rate = YTM (দাম = $১,০০০) |
| **Premium Bond** | Coupon Rate > YTM (দাম > $১,০০০) |
| **Discount Bond** | Coupon Rate < YTM (দাম < $১,০০০) |

**উদাহরণ — বন্ড মূল্যায়ন:**
> ৫ বছর মেয়াদি, ৬% কুপন (বার্ষিক), YTM = ৫%, Par = $১,০০০
>
> কুপনের PV = $৬০ × [১-(১.০৫)⁻⁵] / ০.০৫ = $৬০ × ৪.৩২৯৫ = $২৫৯.৭৭
> Par-এর PV = $১,০০০ / (১.০৫)⁵ = $৭৮৩.৫৩
>
> বন্ডের দাম = $২৫৯.৭৭ + $৭৮৩.৫৩ = **$১,০৪৩.৩০ (Premium Bond)**

---

## 📖 3. Accrued Interest

দুই কুপন পেমেন্টের মধ্যে বন্ড কিনলে জমা সুদ দিতে হবে।

**বন্ডের কোটা মূল্য (Clean Price):** জমা সুদ বাদ দিয়ে
**বন্ডের প্রকৃত মূল্য (Dirty Price):** জমা সুদ সহ

`Dirty Price = Clean Price + Accrued Interest`

## ✍️ অনুশীলন

### প্রশ্ন ১:
৩ বছর মেয়াদি, ৫% কুপন (বার্ষিক), YTM ৬%, Par $১,০০০ → বন্ডের দাম?

**সমাধান:**
কুপনের PV = $৫০ × [১-(১.০৬)⁻³]/০.০৬ = $৫০ × ২.৬৭৩ = $১৩৩.৬৫
Par-এর PV = $১,০০০/(১.০৬)³ = $৮৩৯.৬২
দাম = $১৩৩.৬৫ + $৮৩৯.৬২ = **$৯৭৩.২৭ (Discount Bond)**

### প্রশ্ন ২:
বন্ডের দাম $১,০৫০, কুপন ৭%, Par $১,০০০ — এটি Premium নাকি Discount?

**উত্তর:** Premium Bond (দাম > Par)। YTM < Coupon Rate হবে।
