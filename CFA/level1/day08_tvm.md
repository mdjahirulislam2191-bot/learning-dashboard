# 📘 Day 8: Quant — Time Value of Money (TVM)

## 🎯 শেখার উদ্দেশ্য

এই পাঠ শেষে আপনি যা পারবেন:
1. টাকার সময়মূল্য (TVM) ধারণা ব্যাখ্যা করতে
2. ভবিষ্যৎ মূল্য (FV) ও বর্তমান মূল্য (PV) গণনা করতে
3. সুদের হার (Interest Rate) এর বিভিন্ন প্রকার বুঝতে
4. সাধারণ ও চক্রবৃদ্ধি সুদ গণনা করতে
5. বিভিন্ন ফাইন্যান্সিয়াল ক্যালকুলেশন ব্যবহার করতে

---

## 📖 1. টাকার সময়মূল্য (TVM) কী?

টাকার সময়মূল্য একটি মৌলিক অর্থনৈতিক ধারণা: **আজকের ১ টাকা ভবিষ্যতের ১ টাকার চেয়ে বেশি মূল্যবান।**

**কেন?**
1. **মুদ্রাস্ফীতি (Inflation):** সময়ের সাথে সাথে জিনিসের দাম বাড়ে
2. **সুযোগ ব্যয় (Opportunity Cost):** আজকের টাকা বিনিয়োগ করে রিটার্ন অর্জন করা যায়
3. **ঝুঁকি (Risk):** ভবিষ্যৎ অনিশ্চিত

**ডেটা অ্যানালিস্ট উদাহরণ:**
> আপনি একটি কোম্পানির ডেটা বিশ্লেষণ করছেন। কোম্পানিটি ৫ বছর পর $১০,০০০ পাবে। ১০% ডিসকাউন্ট রেটে, আজকের মূল্য কত? **PV = $১০,০০০ / (১.১০)⁵ = $৬,২০৯।** আজকের $৬,২০৯ ভবিষ্যতের $১০,০০০-এর সমান।

---

## 📖 2. সুদের হার (Interest Rate)

সুদের হার = রিয়েল রিস্ক-ফ্রি রেট + মুদ্রাস্ফীতি প্রিমিয়াম + রিস্ক প্রিমিয়াম

**সুদের হারের উপাদান:**
| উপাদান | ব্যাখ্যা |
|---------|----------|
| রিয়েল রিস্ক-ফ্রি রেট | সময়ের অগ্রাধিকার — ভোগ স্থগিত করার জন্য পুরস্কার |
| মুদ্রাস্ফীতি প্রিমিয়াম | ক্রয়ক্ষমতা হ্রাসের জন্য ক্ষতিপূরণ |
| ডিফল্ট রিস্ক প্রিমিয়াম | খেলাপি হওয়ার ঝুঁকি |
| তরলতা প্রিমিয়াম | দ্রুত বিক্রির অসুবিধার জন্য |
| ম্যাচিউরিটি প্রিমিয়াম | দীর্ঘমেয়াদি বিনিয়োগের ঝুঁকি |

---

## 📖 3. সাধারণ সুদ (Simple Interest)

শুধুমাত্র মূলধনের উপর সুদ গণনা করা হয়।

**সূত্র:** `সুদ = P × r × t`

**উদাহরণ:**
> আপনি $১,০০০ ৫% সরল সুদে ৩ বছরের জন্য বিনিয়োগ করলেন।
> সুদ = $১,০০০ × ০.০৫ × ৩ = $১৫০
> মোট = $১,০০০ + $১৫০ = $১,১৫০

---

## 📖 4. চক্রবৃদ্ধি সুদ (Compound Interest)

সুদের উপর সুদ — টাকা দ্রুত বাড়ে।

**সূত্র:** `FV = PV × (1 + r)ⁿ`

**যেখানে:**
- FV = ভবিষ্যৎ মূল্য
- PV = বর্তমান মূল্য
- r = সুদের হার (দশমিকে)
- n = সময়কাল

**ধাপে ধাপে উদাহরণ:**
> আপনি $১,০০০ ১০% চক্রবৃদ্ধি হারে ৩ বছরের জন্য বিনিয়োগ করলেন।
>
> **বছর ১:** $১,০০০ × ১.১০ = $১,১০০
> **বছর ২:** $১,১০০ × ১.১০ = $১,২১০
> **বছর ৩:** $১,২১০ × ১.১০ = $১,৩৩১
>
> FV = $১,০০০ × (১.১০)³ = $১,০০০ × ১.৩৩১ = $১,৩৩১

### চক্রবৃদ্ধির ফ্রিকোয়েন্সি

বার্ষিক চক্রবৃদ্ধি: FV = PV × (1 + r)ⁿ
অর্ধবার্ষিক: FV = PV × (1 + r/2)^(2n)
ত্রৈমাসিক: FV = PV × (1 + r/4)^(4n)
মাসিক: FV = PV × (1 + r/12)^(12n)

**সাধারণ সূত্র:** `FV = PV × (1 + r/m)^(m×n)`

**উদাহরণ — চক্রবৃদ্ধির প্রভাব:**
> $১,০০০, ১০% হারে ১০ বছরের জন্য:
> - বার্ষিক: $১,০০০ × (১.১০)¹⁰ = $২,৫৯৩.৭৪
> - অর্ধবার্ষিক: $১,০০০ × (১.০৫)²⁰ = $২,৬৫৩.৩০
> - মাসিক: $১,০০০ × (১.০০৮৩৩)¹²⁰ = $২,৭০৭.০৪
> - প্রতিদিন: $১,০০০ × (১.০০০২৭৪)³⁶⁵⁰ = $২,৭১৮.২৮

## 📖 5. বর্তমান মূল্য (Present Value)

ভবিষ্যতের টাকার আজকের মূল্য বের করা।

**সূত্র:** `PV = FV / (1 + r)ⁿ`

**উদাহরণ:**
> ৫ বছর পর $১০,০০০ পাবেন। ৮% ডিসকাউন্ট রেটে আজকের মূল্য কত?
>
> PV = $১০,০০০ / (১.০৮)⁵
> PV = $১০,০০০ / ১.৪৬৯৩
> PV = $৬,৮০৫.৮৩

---

## 📖 6. অ্যানুইটি (Annuity)

অ্যানুইটি হলো সমান সময় অন্তর সমান পরিমাণ অর্থ প্রদান/প্রাপ্তির ধারা।

### সাধারণ অ্যানুইটি (Ordinary Annuity)
প্রতি পিরিয়ডের শেষে পেমেন্ট।

**ভবিষ্যৎ মূল্য:** `FVA = PMT × [(1+r)ⁿ - 1] / r`

**বর্তমান মূল্য:** `PVA = PMT × [1 - (1+r)⁻ⁿ] / r`

### অ্যানুইটি ডিউ (Annuity Due)
প্রতি পিরিয়ডের শুরুতে পেমেন্ট।

**সূত্র:** FV_due = FV_ordinary × (1+r)
PV_due = PV_ordinary × (1+r)

**উদাহরণ — অ্যানুইটি:**
> আপনি ৫ বছর ধরে প্রতি বছর শেষে $১,০০০ জমা করবেন। ৬% সুদে ভবিষ্যৎ মূল্য কত?
>
> FVA = $১,০০০ × [(১.০৬)⁵ - ১] / ০.০৬
> FVA = $১,০০০ × [১.৩৩৮২ - ১] / ০.০৬
> FVA = $১,০০০ × ৫.৬৩৭১
> FVA = $৫,৬৩৭.১০

## ✍️ অনুশীলন (Practice Questions)

### প্রশ্ন ১:
আপনি $৫,০০০ ৮% হারে ৬ বছরের জন্য বিনিয়োগ করলেন। চক্রবৃদ্ধি বার্ষিক হলে FV কত?

**সমাধান:**
FV = $৫,০০০ × (১.০৮)⁶
FV = $৫,০০০ × ১.৫৮৬৯
**FV = $৭,৯৩৪.৩৭**

### প্রশ্ন ২:
১০ বছর পর $২০,০০০ পেতে আজকে কত বিনিয়োগ করতে হবে? সুদের হার ১২% (বার্ষিক চক্রবৃদ্ধি)।

**সমাধান:**
PV = $২০,০০০ / (১.১২)¹⁰
PV = $২০,০০০ / ৩.১০৫৮
**PV = $৬,৪৩৯.৪৬**

### প্রশ্ন ৩:
আপনি প্রতি মাসের শেষে $৫০০ করে ৩ বছর ধরে সঞ্চয় করবেন। মাসিক সুদের হার ০.৫%। ভবিষ্যৎ মূল্য কত?

**সমাধান:**
n = ৩ × ১২ = ৩৬
r = ০.০০৫
FVA = $৫০০ × [(১.০০৫)³⁶ - ১] / ০.০০৫
FVA = $৫০০ × [১.১৯৬৭ - ১] / ০.০০৫
FVA = $৫০০ × ৩৯.৩৩৬১
**FVA = $১৯,৬৬৮.০৫**

## 📝 সূত্র সংক্ষেপ

| ধারণা | সূত্র |
|-------|-------|
| ভবিষ্যৎ মূল্য (FV) | FV = PV × (1 + r)ⁿ |
| বর্তমান মূল্য (PV) | PV = FV / (1 + r)ⁿ |
| সাধারণ সুদ | I = P × r × t |
| অ্যানুইটি FV | FVA = PMT × [(1+r)ⁿ - 1] / r |
| অ্যানুইটি PV | PVA = PMT × [1 - (1+r)⁻ⁿ] / r |
