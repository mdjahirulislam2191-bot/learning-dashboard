# Day 01: ডেটা সায়েন্স ওভারভিউ
## Data Science Overview

### ডেটা সায়েন্স কী?
ডেটা সায়েন্স হলো ডেটা থেকে জ্ঞান ও ইনসাইট বের করার একটি আন্তঃশৃঙ্খল ক্ষেত্র। এটি পরিসংখ্যান, কম্পিউটার সায়েন্স, ডোমেইন নলেজ এবং মেশিন লার্নিং-এর সমন্বয়।

### ডেটা সায়েন্সের প্রধান ধাপসমূহ:
1. **ডেটা সংগ্রহ** (Data Collection) - বিভিন্ন উৎস থেকে ডেটা সংগ্রহ
2. **ডেটা ক্লিনিং** (Data Cleaning) - অনুপস্থিত ও অসামঞ্জস্যপূর্ণ ডেটা পরিষ্কার
3. **Exploratory Data Analysis (EDA)** - ডেটা বোঝা ও ভিজুয়ালাইজ করা
4. **ফিচার ইঞ্জিনিয়ারিং** (Feature Engineering) - উপযুক্ত ফিচার তৈরি
5. **মডেলিং** (Modeling) - মেশিন লার্নিং মডেল তৈরি
6. **ইভালুয়েশন** (Evaluation) - মডেলের পারফরম্যান্স মূল্যায়ন
7. **ডিপ্লয়মেন্ট** (Deployment) - প্রোডাকশনে মডেল স্থাপন

### ডেটা সায়েন্সের টুলস ও লাইব্রেরি:
- **Python**: প্রধান প্রোগ্রামিং ভাষা
- **NumPy**: সংখ্যাত্মক কম্পিউটিং
- **Pandas**: ডেটা ম্যানিপুলেশন
- **Matplotlib/Seaborn**: ভিজুয়ালাইজেশন
- **Scikit-learn**: মেশিন লার্নিং
- **Jupyter Notebook**: ইন্টারেক্টিভ ডেভেলপমেন্ট

```python
# ডেটা সায়েন্সের বেসিক ওয়ার্কফ্লো
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

# স্যাম্পল ডেটা তৈরি
np.random.seed(42)
data = {
    'নাম': ['আলিফ', 'বর্ণা', 'চন্দন', 'দীপা', 'ইমরান'],
    'বয়স': np.random.randint(20, 40, 5),
    'বেতন': np.random.randint(30000, 80000, 5),
    'শহর': ['ঢাকা', 'চট্টগ্রাম', 'খুলনা', 'রাজশাহী', 'সিলেট']
}
df = pd.DataFrame(data)
print("=== ডেটাসেট ===")
print(df)
print("\n=== পরিসংখ্যান ===")
print(df.describe())
print("\n=== ডেটার ধরন ===")
print(df.dtypes)
```

### ডেটা সায়েন্সের প্রয়োগ:
- **হেলথকেয়ার**: রোগ নির্ণয়, ড্রাগ ডিসকভারি
- **ফাইন্যান্স**: ফ্রড ডিটেকশন, রিস্ক অ্যানালাইসিস
- **ই-কমার্স**: রিকমেন্ডেশন সিস্টেম, কাস্টমার সেগমেন্টেশন
- **সোশ্যাল মিডিয়া**: সেন্টিমেন্ট অ্যানালাইসিস, কন্টেন্ট মডারেশন

### সারসংক্ষেপ:
ডেটা সায়েন্স একটি গুরুত্বপূর্ণ ও ক্রমবর্ধমান ক্ষেত্র যা ডেটা ব্যবহার করে সিদ্ধান্ত গ্রহণে সাহায্য করে। এই কোর্সে আমরা ৩০ দিনে ডেটা সায়েন্সের সব মৌলিক ও উন্নত বিষয় শিখব।
