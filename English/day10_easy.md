## 📝 Using Technical Vocabulary in Interviews

### Sample Answer 1
**Question:** "What SQL commands do you use most often?"
**Answer:** "I frequently use SELECT with WHERE to filter data. I also use JOIN to combine tables — especially INNER JOIN and LEFT JOIN. For summaries, I use GROUP BY with aggregate functions like COUNT and SUM."

### Sample Answer 2
**Question:** "How do you ensure data accuracy?"
**Answer:** "I always validate my data. First, I check for null values and duplicates. Then I verify that the row counts match expectations. I also cross-check a sample of my results against the source data."

## 📖 Grammar Point: Using Technical Words in Sentences
When using technical terms, explain them briefly so the interviewer knows you understand.

**BAD:** "I used ETL." (too simple)
**GOOD:** "I used an ETL process — Extract, Transform, Load — to move data from the source system into our data warehouse."

## 💡 Interview Tip
**Don't pretend to know tools you don't.** If you are asked about a tool you haven't used, say: "I haven't worked with that tool yet, but I have experience with similar tools and I can learn it quickly."
