# দিন ৯: অ্যাডভান্সড ফর্মুলা (SUMIFS, COUNTIFS, অ্যারে)

## 🎯 আজকের লক্ষ্য
Excel-এর শর্তাধীন যোগ/গণনা ফর্মুলা এবং অ্যারে ফর্মুলা ব্যবহার করে জটিল বিশ্লেষণ শেখা।

## 📊 SUMIFS ফাংশন

### সিনট্যাক্স
```excel
=SUMIFS(যোগ_কলাম, শর্ত_কলাম১, শর্ত১, [শর্ত_কলাম২, শর্ত২], ...)
```

### প্যারামিটার
| প্যারামিটার | ব্যাখ্যা |
|-------------|---------|
| sum_range | যে কলামের মান যোগ হবে |
| criteria_range1 | প্রথম শর্তের কলাম |
| criteria1 | প্রথম শর্ত |
| criteria_range2 | দ্বিতীয় শর্তের কলাম (অপশনাল) |
| criteria2 | দ্বিতীয় শর্ত (অপশনাল) |

### 💼 ফাইন্যান্স উদাহরণ

#### উদাহরণ ১: নির্দিষ্ট ক্যাটাগরির যোগফল
```excel
// অফিস সাপ্লাই ক্যাটাগরির মোট খরচ
=SUMIFS(E:E, C:C, "অফিস সাপ্লাই")

// ২০২৪ সালের জানুয়ারির বিক্রয়
=SUMIFS(D:D, A:A, ">=01-01-2024", A:A, "<=31-01-2024")
```

#### উদাহরণ ২: একাধিক শর্ত
```excel
// ঢাকা অঞ্চলে ল্যাপটপের মোট বিক্রয়
=SUMIFS(বিক্রয়_কলাম, অঞ্চল_কলাম, "ঢাকা", পণ্য_কলাম, "ল্যাপটপ")

// ২০২৪ সালে ১০০০০+ মূল্যের পণ্যের বিক্রয়
=SUMIFS(বিক্রয়_কলাম, বছর_কলাম, ২০২৪, মূল্য_কলাম, ">=10000")
```

#### উদাহরণ ৩: ওয়াইল্ডকার্ড সহ
```excel
// 'ল্যাপ' দিয়ে শুরু সব পণ্যের বিক্রয়
=SUMIFS(E:E, C:C, "ল্যাপ*")

// 'ফোন' দিয়ে শেষ সব পণ্যের বিক্রয়
=SUMIFS(E:E, C:C, "*ফোন")
```

## 🔢 COUNTIFS ফাংশন

### সিনট্যাক্স
```excel
=COUNTIFS(শর্ত_কলাম১, শর্ত১, [শর্ত_কলাম২, শর্ত২], ...)
```

### 💼 ফাইন্যান্স উদাহরণ

#### উদাহরণ ১
```excel
// ঢাকা অঞ্চলের ক্লায়েন্ট সংখ্যা
=COUNTIFS(অঞ্চল_কলাম, "ঢাকা")

// ২০২৪ সালের ১০০০০+ বিক্রির সংখ্যা
=COUNTIFS(বিক্রয়_কলাম, ">=10000", বছর_কলাম, ২০২৪)
```

#### উদাহরণ ২: শর্তাধীন গণনা
```excel
// পেমেন্ট বাকি আছে এমন ইনভয়েস সংখ্যা
=COUNTIFS(স্টেটাস_কলাম, "বাকি")

// ৬০+ দিন ওভারডিউ অ্যাকাউন্ট
=COUNTIFS(দিন_কলাম, ">60", স্টেটাস_কলাম, "পেমেন্ট বাকি")
```

## 📋 AVERAGEIFS, MAXIFS, MINIFS

```excel
// বিভাগ অনুযায়ী গড় বিক্রয়
=AVERAGEIFS(বিক্রয়_কলাম, বিভাগ_কলাম, "ইলেকট্রনিক্স")

// ঢাকা অঞ্চলের সর্বোচ্চ বিক্রয়
=MAXIFS(বিক্রয়_কলাম, অঞ্চল_কলাম, "ঢাকা")

// জানুয়ারির সর্বনিম্ন বিক্রয়
=MINIFS(বিক্রয়_কলাম, মাস_কলাম, "জানুয়ারি")
```

## ⚡ অ্যারে ফর্মুলা (Array Formulas)

### অ্যারে ফর্মুলা কী?
একটি ফর্মুলা যা একাধিক মান নিয়ে কাজ করে এবং একাধিক ফলাফল দিতে পারে।

### একক সেল অ্যারে ফর্মুলা
```excel
// মোট বিক্রয় যেখানে পণ্য=ল্যাপটপ AND অঞ্চল=ঢাকা
{=SUM(IF((A2:A100="ল্যাপটপ")*(B2:B100="ঢাকা"), C2:C100))}
// পুরনো Excel-এ Ctrl+Shift+Enter দিতে হবে
// নতুন Excel-এ সরাসরি Enter
```

### একাধিক সেল অ্যারে ফর্মুলা
```excel
// একবারে একাধিক ফলাফল
{=A2:A10*B2:B10}  → প্রতিটি সেলে গুণফল
// একটি রেঞ্জ সিলেক্ট করে ফর্মুলা লিখুন, তারপর Ctrl+Shift+Enter
```

## 💼 অ্যাডভান্সড ফাইন্যান্স ফর্মুলা

### ১. ট্রান্সপোজ (TRANSPOSE)
```excel
// সারিকে কলামে বা কলামকে সারিতে রূপান্তর
{=TRANSPOSE(A1:D1)}
```

### ২. ফ্রিকোয়েন্সি (FREQUENCY)
```excel
// ডেটার ফ্রিকোয়েন্সি ডিস্ট্রিবিউশন
{=FREQUENCY(বিক্রয়_ডেটা, বিন_রেঞ্জ)}
```

### ৩. যোগফলের জটিল শর্ত
```excel
// SUMPRODUCT ব্যবহার (অ্যারে ফর্মুলা নয়)
=SUMPRODUCT((A2:A100="ঢাকা")*(B2:B100="ল্যাপটপ")*C2:C100)

// একাধিক কলামের শর্ত
=SUMPRODUCT((A2:A100="ঢাকা")*(MONTH(B2:B100)=1)*C2:C100)
```

## 🧩 LET ও LAMBDA ফাংশন (আধুনিক)

### LET ফাংশন
ভেরিয়েবল ডিফাইন করে ফর্মুলা সহজ করা:
```excel
=LET(
    বিক্রয়, D2:D100,
    খরচ, E2:E100,
    লাভ, SUM(বিক্রয়)-SUM(খরচ),
    লাভ
)
```

### LAMBDA ফাংশন
কাস্টম ফাংশন তৈরি:
```excel
// লাভ মার্জিন ফাংশন
=LAMBDA(বিক্রয়, খরচ, (বিক্রয়-খরচ)/বিক্রয়)

// নামযুক্ত রেঞ্জে সংরক্ষণ:
Formulas → Name Manager → New
Name: মার্জিন
Refers to: =LAMBDA(বিক্রয়, খরচ, (বিক্রয়-খরচ)/বিক্রয়)

// ব্যবহার:
=মার্জিন(D2, E2)
```

## 💼 ফাইন্যান্স কেস স্টাডি

### কেস: মাসিক বিক্রয় রিপোর্ট
```excel
ডেটা:
A: তারিখ | B: পণ্য | C: অঞ্চল | D: বিক্রয় | E: খরচ

প্রয়োজনীয় ফর্মুলা:

১. জানুয়ারির মোট বিক্রয়:
=SUMIFS(D:D, A:A, ">=01-01-2024", A:A, "<=31-01-2024")

২. ইলেকট্রনিক্স ক্যাটাগরির গড় লাভ মার্জিন:
=AVERAGEIFS((D:D-E:E)/D:D, B:B, "ইলেকট্রনিক্স")

৩. সব অঞ্চলের জন্য বিক্রয়কর্মী প্রতি গড় বিক্রয়:
=SUMPRODUCT((C:C=F2)*D:D)/COUNTIFS(C:C, F2)

৪. প্রতি মাসের মোট বিক্রয় (একটি ফর্মুলায়):
{=SUMIFS(D:D, A:A, ">="&G2:G13, A:A, "<="&EOMONTH(G2:G13,0))}
```

## ✅ প্র্যাকটিস টাস্ক
1. SUMIFS ব্যবহার করে বিভাগ ও মাস অনুযায়ী বিক্রয় যোগ করুন
2. COUNTIFS দিয়ে নির্দিষ্ট শর্তে লেনদেন সংখ্যা গণনা করুন
3. AVERAGEIFS দিয়ে প্রতিটি পণ্যের গড় বিক্রয় বের করুন
4. SUMPRODUCT দিয়ে ওয়েটেড এভারেজ বের করুন
5. একটি অ্যারে ফর্মুলা তৈরি করুন যা দুই শর্তে যোগফল দেয়
6. LET ফাংশন ব্যবহার করে একটি জটিল ফর্মুলা সহজ করুন

## 📝 টিপস
- SUMIFS-এ প্রথম প্যারামিটার যোগ_কলাম -> পরে শর্ত
- COUNTIFS-এ সব প্যারামিটারই শর্ত (কোনো যোগ_কলাম নেই)
- SUMPRODUCT অ্যারে ফর্মুলার চেয়ে দ্রুত এবং সহজ
- আধুনিক Excel-এ অ্যারে ফর্মুলার জন্য Ctrl+Shift+Enter লাগে না
- বড় ডেটাসেটে SUMPRODUCT ধীর হতে পারে — SUMIFS ব্যবহার করুন
- নামযুক্ত রেঞ্জ (Named Range) ব্যবহার করলে ফর্মুলা বোধগম্য হয়
