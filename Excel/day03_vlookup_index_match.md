# দিন ৩: VLOOKUP, INDEX-MATCH (লুকআপ ফাংশন)

## 🎯 আজকের লক্ষ্য
Excel-এর সবচেয়ে গুরুত্বপূর্ণ লুকআপ ফাংশন VLOOKUP ও INDEX-MATCH শেখা — ডেটা খোঁজা ও মিলানোর জন্য।

## 🔍 VLOOKUP ফাংশন

### সিনট্যাক্স
```excel
=VLOOKUP(লুকআপ_ভ্যালু, টেবিল_অ্যারে, কলাম_নম্বর, [রেঞ্জ_লুকআপ])
```

### প্যারামিটার ব্যাখ্যা
| প্যারামিটার | বর্ণনা |
|-------------|--------|
| lookup_value | যে মান খুঁজবেন (যেমন পণ্য কোড) |
| table_array | ডেটার রেঞ্জ (প্রথম কলামে lookup_value থাকতে হবে) |
| col_index_num | যে কলাম থেকে মান আনবেন (১ থেকে শুরু) |
| range_lookup | FALSE = exact match, TRUE = approximate |

### ফাইন্যান্স উদাহরণ
```excel
// পণ্য কোড দিয়ে দাম খোঁজা
=VLOOKUP("P1001", A2:C100, 3, FALSE)

// ক্লায়েন্ট আইডি দিয়ে ব্যালেন্স দেখা
=VLOOKUP(B5, ক্লায়েন্ট_ডেটা!A:D, 4, FALSE)
```

### ⚠️ VLOOKUP-এর সীমাবদ্ধতা
- ❌ শুধু ডানদিকে খুঁজতে পারে (লুকআপ কলাম বামে থাকতে হবে)
- ❌ এক কলাম মুছে গেলে ফলাফল ভুল হয়
- ❌ বড় ডেটাসেটে ধীরগতি

## 🔗 INDEX-MATCH (VLOOKUP-এর বিকল্প)

### INDEX ফাংশন
```excel
=INDEX(অ্যারে, সারি_নম্বর, [কলাম_নম্বর])
```

### MATCH ফাংশন
```excel
=MATCH(লুকআপ_ভ্যালু, লুকআপ_অ্যারে, [ম্যাচ_টাইপ])
```
- match_type: 0=exact, 1=less than, -1=greater than

### INDEX + MATCH কম্বিনেশন
```excel
=INDEX(রিটার্ন_কলাম, MATCH(লুকআপ_ভ্যালু, লুকআপ_কলাম, 0))
```

### ফাইন্যান্স উদাহরণ
```excel
// ক্লায়েন্ট নাম দিয়ে ব্যালেন্স বের করা (VLOOKUP-এর চেয়ে নমনীয়)
=INDEX(D:D, MATCH("জন_ডো", A:A, 0))

// দুই শর্তে ডেটা খোঁজা
=INDEX(C2:C100, MATCH(1, (A2:A100="পণ্য_X")*(B2:B100="ঢাকা"), 0))
```

## 💼 ফাইন্যান্স কেস স্টাডি

### পরিস্থিতি ১: ইনভয়েস ম্যাচিং
```
শীট ১: বিক্রয় রেকর্ড (ইনভয়েস নং, তারিখ, ক্লায়েন্ট)
শীট ২: পেমেন্ট রেকর্ড (ইনভয়েস নং, পেমেন্ট তারিখ, পরিমাণ)

=VLOOKUP(A2, পেমেন্ট!A:C, 3, FALSE)
// এটা দেখাবে পেমেন্ট হয়েছে কি না
```

### পরিস্থিতি ২: এক্সচেঞ্জ রেট কনভার্শন
```excel
// মুদ্রা কোড দিয়ে এক্সচেঞ্জ রেট আনা
=INDEX(রেট_কলাম, MATCH("USD", মুদ্রা_কলাম, 0)) * বিক্রয়_পরিমাণ
```

### পরিস্থিতি ৩: দুই কলামের অনন্য কম্বিনেশন
```excel
// = বছর ও মাস অনুযায়ী ডেটা খোঁজা
{=INDEX(বিক্রয়, MATCH(1, (বছর=২০২৪)*(মাস="জানুয়ারি"), 0))}
// এটি CSE অ্যারে ফর্মুলা (Ctrl+Shift+Enter)
```

## 🆚 VLOOKUP vs INDEX-MATCH তুলনা

| বৈশিষ্ট্য | VLOOKUP | INDEX-MATCH |
|-----------|---------|-------------|
| ব্যবহার সহজতর | ✅ | ❌ |
| বাম দিকে খোঁজা | ❌ | ✅ |
| ডান দিকে খোঁজা | ✅ | ✅ |
| কলাম যোগ/মুছে ফেলা | ❌ (ভুল হয়) | ✅ (সঠিক) |
| গতি (বড় ডেটা) | মাঝারি | দ্রুত |
| দুই শর্তে খোঁজা | ❌ | ✅ |

## ✅ প্র্যাকটিস টাস্ক
1. একটি ডেটাসেট তৈরি করুন:
   - ২০টি পণ্যের আইডি, নাম, বিভাগ, দাম, স্টক
2. VLOOKUP ব্যবহার করে পণ্য আইডি দিয়ে দাম বের করুন
3. INDEX-MATCH ব্যবহার করে পণ্য নাম দিয়ে বিভাগ বের করুন
4. দুইটি টেবিলের ডেটা ম্যাচ করে দেখান
5. একটি এক্সচেঞ্জ রেট ক্যালকুলেটর তৈরি করুন

## 📝 টিপস
- পরিষ্কার ডেটার জন্য সবসময় FALSE (exact match) ব্যবহার করুন
- টেবিল অ্যারেতে $ দিয়ে Absolute Reference দিন
- নামযুক্ত রেঞ্জ (Named Range) ব্যবহার করলে ফর্মুলা পড়া সহজ হয়
- **IFERROR** দিয়ে এরর হ্যান্ডেল করুন: `=IFERROR(VLOOKUP(...), "পাওয়া যায়নি")`
