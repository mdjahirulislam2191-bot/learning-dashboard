## 📁 স্টেপ ১: ডেটা তৈরি

### ডেটাসেট: বিক্রয় ও আর্থিক তথ্য (২০২৪)

নিচের কলামগুলো নিয়ে একটি Excel শীট তৈরি করুন:

```excel
কলামসমূহ:
A: লেনদেন আইডি  (TXN-0001 থেকে TXN-1000)
B: তারিখ         (০১-০১-২০২৪ থেকে ৩১-১২-২০২৪)
C: পণ্য বিভাগ    (ইলেকট্রনিক্স, ক্লোথিং, ফুড, ফার্নিচার, বুকস)
D: পণ্যের নাম    (প্রতিটি বিভাগে ৫টি করে পণ্য)
E: অঞ্চল         (ঢাকা, চট্টগ্রাম, রাজশাহী, খুলনা, সিলেট)
F: বিক্রয়কর্মী  (১০ জনের নাম)
G: পরিমাণ        (১-১০০)
H: ইউনিট মূল্য   (১০০-৫০০০)
I: মোট বিক্রয়   (=G*H)
J: খরচ           (=I*০.৬ থেকে I*০.৮ - র‍্যান্ডম)
K: লাভ           (=I-J)
L: লাভ মার্জিন   (=K/I)
```

### ডেটা জেনারেট করার জন্য ফর্মুলা:
```excel
A2: ="TXN-"&TEXT(ROW()-1,"0000")
B2: =RANDARRAY(1000,1,DATE(2024,1,1),DATE(2024,12,31),TRUE)
C2: =INDEX(বিভাগ_লিস্ট, RANDBETWEEN(1,5))
D2: =INDEX(পণ্য_লিস্ট, RANDBETWEEN(1,25))
E2: =INDEX(অঞ্চল_লিস্ট, RANDBETWEEN(1,5))
F2: =INDEX(কর্মী_লিস্ট, RANDBETWEEN(1,10))
G2: =RANDBETWEEN(1,100)
H2: =RANDBETWEEN(100,5000)
I2: =G2*H2
J2: =ROUND(I2*RANDBETWEEN(60,80)/100,0)
K2: =I2-J2
L2: =K2/I2
```

## 📊 স্টেপ ২: Pivot Table বিশ্লেষণ

### আলাদা শীটে Pivot Tables তৈরি করুন:

#### PT1: বিভাগ ও মাসভিত্তিক বিক্রয়
```
Rows: পণ্য বিভাগ, Columns: মাস (Group by Month)
Values: মোট বিক্রয় (Sum)
```

#### PT2: অঞ্চল ও বিক্রয়কর্মীভিত্তিক পারফরম্যান্স
```
Rows: অঞ্চল, Columns: বিক্রয়কর্মী
Values: মোট লাভ (Sum)
```

#### PT3: পণ্য পারফরম্যান্স
```
Rows: পণ্যের নাম
Values: মোট বিক্রয়, মোট লাভ, লাভ মার্জিন (Average)
Filter: Top 10 by বিক্রয়
```

## 📈 স্টেপ ৩: KPI ও ফর্মুলা

### মূল KPI হিসাব করুন:

```excel
KPI ১: মোট রাজস্ব (YTD)
=SUM(ডেটা[মোট_বিক্রয়])

KPI ২: মোট খরচ
=SUM(ডেটা[খরচ])

KPI ৩: নিট লাভ
=SUM(ডেটা[লাভ])

KPI ৪: লাভ মার্জিন (%)
=KPI৩/KPI১

KPI ৫: গড় লেনদেনের পরিমাণ
=AVERAGE(ডেটা[মোট_বিক্রয়])

KPI ৬: মোট লেনদেন সংখ্যা
=COUNTA(ডেটা[লেনদেন_আইডি])

KPI ৭: সর্বোচ্চ বিক্রির দিন
=MAX(SUMIFS(ডেটা[মোট_বিক্রয়],ডেটা[তারিখ],তারিখ_কলাম))

KPI ৮: MoM Growth Rate
=(বর্তমান_মাস_বিক্রয়-পূর্ববর্তী_মাস_বিক্রয়)/পূর্ববর্তী_মাস_বিক্রয়
```

### Conditional Formatting সহ:
```excel
লাভ মার্জিন:
  >= ২৫% → সবুজ ব্যাকগ্রাউন্ড
  >= ১৫% → হলুদ ব্যাকগ্রাউন্ড
  < ১৫%  → লাল ব্যাকগ্রাউন্ড
```

## 📉 স্টেপ ৬: What-If Analysis

### গোল সিক সেকশন:
```excel
প্রশ্ন: ৩০% লাভ মার্জিন পেতে কত বিক্রয় দরকার?

সেটআপ:
B1: বর্তমান বিক্রয় = ২৫০০০০০
B2: বর্তমান খরচ = ১৮০০০০০
B3: বর্তমান মার্জিন = (B1-B2)/B1
B4: টার্গেট মার্জিন = ৩০%

গোল সিক: Set B3=0.30, By changing B1
```

### সিনারিও ম্যানেজার:
```excel
সিনারিও ১: "বেস্ট কেস" (বিক্রয় +২০%, খরচ +৫%)
সিনারিও ২: "ওয়ার্স্ট কেস" (বিক্রয় -১০%, খরচ +১৫%)
সিনারিও ৩: "মোস্ট লাইকলি" (বিক্রয় +১০%, খরচ +৮%)
```

## 🤖 স্টেপ ৭: অটোমেশন (VBA)

### ম্যাক্রো ১: ড্যাশবোর্ড রিফ্রেশ
```vba
Sub রিফ্রেশ_ড্যাশবোর্ড()
    ' সব Pivot Table রিফ্রেশ
    Dim pt As PivotTable
    For Each pt In ActiveSheet.PivotTables
        pt.RefreshTable
    Next pt
    
    ' সব কানেকশন রিফ্রেশ
    ThisWorkbook.RefreshAll
    
    MsgBox "ড্যাশবোর্ড আপডেট করা হয়েছে!", vbInformation
End Sub
```

### ম্যাক্রো ২: PDF এক্সপোর্ট
```vba
Sub PDF_এক্সপোর্ট()
    Dim পাথ As String
    পাথ = ThisWorkbook.Path & "\Dashboard_Report_" & Format(Date, "yyyymmdd") & ".pdf"
    
    Sheets("Dashboard").ExportAsFixedFormat _
        Type:=xlTypePDF, _
        Filename:=পাথ, _
        Quality:=xlQualityStandard
    
    MsgBox "PDF রিপোর্ট তৈরি হয়েছে: " & পাথ, vbInformation
End Sub
```
