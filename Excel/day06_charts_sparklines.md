# দিন ৬: Charts & Sparklines (ডেটা ভিজুয়ালাইজেশন)

## 🎯 আজকের লক্ষ্য
Excel-এ পেশাদার চার্ট ও স্পার্কলাইন তৈরি করে ডেটা ভিজুয়ালাইজেশন শেখা — ফাইন্যান্স রিপোর্টের জন্য।

## 📊 চার্টের প্রকারভেদ

### ১. কলাম চার্ট (Column Chart)
```excel
উপযোগিতা: বিভিন্ন বিভাগের তুলনা
ফাইন্যান্স: মাসিক বিক্রয়, বিভাগভিত্তিক খরচ
```
- **ক্লাস্টারড কলাম**: বিভিন্ন বিভাগের পাশাপাশি তুলনা
- **স্ট্যাকড কলাম**: মোটের অংশ হিসেবে প্রদর্শন
- **১০০% স্ট্যাকড কলাম**: শতাংশ ভিত্তিক তুলনা

### ২. লাইন চার্ট (Line Chart)
```excel
উপযোগিতা: সময়ের সাথে পরিবর্তন দেখানো
ফাইন্যান্স: স্টক প্রাইস, বিক্রয় ট্রেন্ড, GDP গ্রোথ
```
- টাইম সিরিজ ডেটার জন্য সেরা
- একাধিক সিরিজ যোগ করা যায়

### ৩. পাই চার্ট (Pie Chart)
```excel
উপযোগিতা: মোটের শতাংশ ভাগ দেখানো
ফাইন্যান্স: বাজেট বরাদ্দ, খরচের ক্যাটাগরি
```
- ⚠️ সর্বোচ্চ ৫-৭টি বিভাগের জন্য ব্যবহার করুন
- ডোনাট চার্ট (Doughnut) আধুনিক বিকল্প

### ৪. বার চার্ট (Bar Chart)
```excel
উপযোগিতা: অনুভূমিক তুলনা (লম্বা লেবেলের জন্য)
ফাইন্যান্স: পণ্য বিক্রয় র্যাঙ্কিং
```

### ৫. কম্বো চার্ট (Combo Chart)
```excel
উপযোগিতা: দুই ধরনের ডেটা একসাথে
ফাইন্যান্স: বিক্রয় (কলাম) + লাভ মার্জিন (লাইন)
```

## 🎨 প্রফেশনাল চার্ট ডিজাইন

### চার্ট এলিমেন্ট যোগ করা
```
Chart Elements (+ বাটন):
✅ Axis Titles    → অক্ষের নাম
✅ Chart Title    → চার্টের শিরোনাম  
✅ Data Labels    → ডেটার মান
✅ Legend         → সিরিজের নাম
✅ Gridlines      → গ্রিডলাইন
✅ Trendline      → ট্রেন্ড লাইন
✅ Error Bars     → এরর বার
```

### চার্ট স্টাইল
```
Chart Design → Chart Styles:
- Style 1-48 (প্রি-ডিফাইন্ড থিম)
- Colorful (উজ্জ্বল রঙ)
- Monochromatic (একরঙা প্রফেশনাল)
```

### ফাইন্যান্স চার্টের সেরা প্র্যাকটিস
- ✅ ন্যূনতম রঙ ব্যবহার করুন (২-৩ টি)
- ✅ ডেটা লেবেল পরিষ্কার রাখুন
- ✅ তিনটির বেশি সিরিজ না দেখানো
- ✅ Y-অক্ষ শূন্য থেকে শুরু করা
- ✅ গ্রিডলাইন হালকা রাখুন

## 📈 ট্রেন্ডলাইন (Trendline)

### ট্রেন্ডলাইন অপশন
```
Linear        → সরল রৈখিক প্রবণতা
Exponential   → দ্রুত বৃদ্ধি/পতন
Moving Average → গড় প্রবণতা (৭ দিনের)
```

### ফাইন্যান্স ব্যবহার:
```excel
বিক্রয় ডেটায় ট্রেন্ডলাইন যোগ:
Chart → Add Element → Trendline → Linear

ট্রেন্ডলাইন অপশন:
- Display Equation on chart
- Display R-squared value
- Forecast (forward/backward)
```

## 📉 স্পার্কলাইন (Sparklines)

### স্পার্কলাইন কী?
ছোট, সেলের ভিতরের মিনি-চার্ট — বড় চার্টের বিকল্প নয়, পরিপূরক।

### প্রকারভেদ
```
Line     → রেখা চিত্র (ট্রেন্ডের জন্য)
Column   → কলাম চিত্র (তুলনার জন্য)
Win/Loss → পজিটিভ/নেগেটিভ দেখানো
```

### তৈরি করার নিয়ম
```excel
1. যে সেলে স্পার্কলাইন চান সেটি সিলেক্ট করুন
2. Insert → Sparklines
3. ডেটা রেঞ্জ সিলেক্ট করুন
4. Location Range নির্ধারণ করুন
```

### স্পার্কলাইন ফরম্যাটিং
```
Sparkline → Design Tab:
- Marker Color (হাইলাইট পয়েন্ট)
- High/Low Point (সর্বোচ্চ/নিম্ন)
- First/Last Point
- Negative Point (নেতিবাচক মান)
- Line Weight (লাইনের মোটা)
```

### 💼 ফাইন্যান্সে স্পার্কলাইন উদাহরণ
```excel
প্রতিটি পণ্যের পাশে মাসিক বিক্রয় ট্রেন্ড:
পণ্য  | জানু | ফেব্রু | মার্চ | এপ্রিল | ট্রেন্ড
ল্যাপটপ| ৫০   | ৬৫    | ৭০   | ৮৫    | ╱╱╱ (স্পার্কলাইন)
মোবাইল | ৪০   | ৩৫    | ৪৫   | ৩০    | ╲╱╲ (স্পার্কলাইন)
```

## 💼 ফাইন্যান্স চার্ট কেস স্টাডি

### ১. মাসিক রাজস্ব ও খরচ রিপোর্ট
```excel
ডেটা: মাস ১২টি, রাজস্ব ও খরচ
চার্ট: কম্বো চার্ট
- কলাম: রাজস্ব ও খরচ
- লাইন: লাভ মার্জিন (%)
- দ্বিতীয় Y-axis: লাভ মার্জিনের জন্য
```

### ২. পোর্টফোলিও পারফরম্যান্স
```excel
ডেটা: মাসিক রিটার্ন ১২ মাস
চার্ট: লাইন চার্ট
- বেঞ্চমার্ক (S&P 500) সাথে তুলনা
- নেগেটিভ মাস লাল, পজিটিভ মাস সবুজ
- Moving Average ট্রেন্ডলাইন
```

### ৩. বিক্রয় ড্যাশবোর্ড
```excel
এক পৃষ্ঠায়:
- বিক্রয় প্রবণতা (লাইন চার্ট)
- বিভাগভিত্তিক বিক্রয় (পাই/ডোনাট)
- বিক্রয়কর্মী র‍্যাঙ্কিং (বার চার্ট)
- প্রতিটি পণ্যের স্পার্কলাইন
```

## ✅ প্র্যাকটিস টাস্ক
1. মাসিক বিক্রয় ডেটা দিয়ে একটি লাইন চার্ট তৈরি করুন
2. বিভাগভিত্তিক খরচের পাই চার্ট তৈরি করুন
3. বিক্রয় ও খরচের কম্বো চার্ট তৈরি করুন
4. প্রতিটি পণ্যের পাশে স্পার্কলাইন যোগ করুন
5. ট্রেন্ডলাইন ও R-squared দেখান
6. চার্টে ডেটা লেবেল ও টাইটেল যোগ করুন

- চার্ট থিম পরিবর্তন করতে **Page Layout → Themes**
- 3D ইফেক্ট এড়িয়ে চলুন — প্রফেশনাল দেখায় না
