# দিন ১১: ড্যাশবোর্ড ডিজাইন (Dashboard Design in Excel)

## 🎯 আজকের লক্ষ্য
Excel-এ প্রফেশনাল ফাইন্যান্স ড্যাশবোর্ড ডিজাইন করা — যা থেকে এক নজরে সব গুরুত্বপূর্ণ তথ্য বোঝা যায়।

## 📊 ড্যাশবোর্ড কী?
এক পৃষ্ঠায় ডেটার ভিজুয়াল সারসংক্ষেপ — KPI, চার্ট ও গুরুত্বপূর্ণ মেট্রিক্সের সমন্বয়।

## 🏗️ ড্যাশবোর্ড ডিজাইনের মূলনীতি

### ১. পরিচ্ছন্নতা (Clarity)
- ✅ এক নজরে বোধগম্য
- ✅ অপ্রয়োজনীয় তথ্য বাদ
- ✅ লজিক্যাল লেআউট
- ❌ অতিরিক্ত রঙ ও ইফেক্ট নয়

### ২. প্রাসঙ্গিকতা (Relevance)
- ✅ শুধু গুরুত্বপূর্ণ KPI
- ✅ লক্ষ্য দর্শকের প্রয়োজন অনুযায়ী
- ✅ অ্যাকশনযোগ্য তথ্য

### ৩. ধারাবাহিকতা (Consistency)
- ✅ একই থিম ও রঙের স্কিম
- ✅ একই ফন্ট ও সাইজ
- ✅ ইউনিফর্ম স্টাইল

## 🎨 ড্যাশবোর্ড এলিমেন্ট

### KPI কার্ড (Key Performance Indicators)
```excel
উদাহরণ ফাইন্যান্স KPI:
┌──────────────────────────────┐
│  📈 মোট রাজস্ব               │
│  $১,২৫০,০০০                 │
│  ↑ ১২% vs গত মাস            │
└──────────────────────────────┘
```

### সাধারণ ফাইন্যান্স KPI
| KPI | ফর্মুলা | ব্যাখ্যা |
|-----|---------|----------|
| মোট রাজস্ব | =SUM(বিক্রয়) | সর্বমোট আয় |
| লাভ মার্জিন | =(রাজস্ব-খরচ)/রাজস্ব | লাভের হার |
| গ্রোথ রেট | =(বর্তমান-পূর্ব)/পূর্ব | প্রবৃদ্ধি |
| CAC | =মোট_মার্কেটিং_খরচ/নতুন_ক্লায়েন্ট | ক্লায়েন্ট অর্জন খরচ |
| CLV | =গড়_বিক্রয়×ক্রয়_ফ্রিকোয়েন্সি×আয়ুষ্কাল | ক্লায়েন্ট লাইফটাইম ভ্যালু |

## 📐 ড্যাশবোর্ড লেআউট

### গ্রিড লেআউট (১২-কলাম গ্রিড)
```excel
┌──────┬──────┬──────┬──────┐
│ KPI1 │ KPI2 │ KPI3 │ KPI4 │  ← সারি ১: KPI
├──────┴──────┴──────┴──────┤
│      লাইন চার্ট (ট্রেন্ড)     │  ← সারি ২: প্রধান চার্ট
├──────┬──────┬───────────────┤
│ পাই  │ বার  │   টেবিল       │  ← সারি ৩: সাপোর্টিং
│চার্ট │ চার্ট│   (ডেটা)      │
└──────┴──────┴───────────────┘
```

## 💼 ফাইন্যান্স ড্যাশবোর্ড কম্পোনেন্ট

### ১. KPI সারি
```excel
৪টি KPI পাশাপাশি:
- মোট রাজস্ব (YTD)
- মোট খরচ (YTD)  
- নিট লাভ (YTD)
- লাভ মার্জিন (%)
প্রতিটির সাথে গত মাস/গত বছরের তুলনা
```

### ২. ট্রেন্ড চার্ট
```excel
লাইন/এরিয়া চার্ট:
- মাসিক রাজস্ব (১২ মাস)
- Actual vs Target
- Moving Average
- ফোরকাস্ট লাইন
```

### ৩. ক্যাটাগরি বিশ্লেষণ
```excel
পাই/ডোনাট চার্ট:
- পণ্য ক্যাটাগরি অনুযায়ী বিক্রয়
- অঞ্চল অনুযায়ী বিক্রয়

বার চার্ট:
- Top 5 পণ্য
- Bottom 5 পণ্য
- বিভাগভিত্তিক খরচ
```

### ৪. ইন্টারঅ্যাক্টিভ এলিমেন্ট
```excel
স্লাইসার:
- বছর
- অঞ্চল
- পণ্য ক্যাটাগরি

টাইমলাইন:
- মাসিক/ত্রৈমাসিক ভিউ
```

### ৫. ডেটা টেবিল
```excel
সংক্ষিপ্ত টেবিল:
- Conditional Formatting সহ
- Top N আইটেম
- স্পার্কলাইন সহ
```

## 🛠️ ড্যাশবোর্ড তৈরির ধাপ

### ধাপ ১: ডেটা প্রস্তুতি
```excel
- আলাদা শীটে র ডেটা রাখুন
- Power Query দিয়ে ক্লিন করুন
- Named Ranges তৈরি করুন
- Pivot Tables প্রস্তুত করুন
```

### ধাপ ২: লেআউট ডিজাইন
```excel
1. হেডার সারি (কোম্পানির নাম, সময়কাল)
2. KPI সারি (বড় ফন্ট, কনডিশনাল কালার)
3. চার্ট এরিয়া (গ্রিডে সাজানো)
4. ফিল্টার সেকশন (স্লাইসার)
5. ফুটার (দ্রষ্টব্য, তারিখ)
```

### ধাপ ৩: ফরম্যাটিং
```excel
কালার থিম:
- হেডার: গাঢ় নীল (#1F4E79)
- পজিটিভ: সবুজ (#00B050)
- নেগেটিভ: লাল (#FF0000)
- ব্যাকগ্রাউন্ড: সাদা/হালকা ধূসর
- ফন্ট: Calibri/Segoe UI

বর্ডার:
- হালকা গ্রিডলাইন
- সেকশনের জন্য ডিভাইডার
```

### ধাপ ৪: ইন্টারঅ্যাক্টিভিটি
```excel
- স্লাইসার কানেক্ট করুন (Report Connections)
- টাইমলাইন যোগ করুন
- হাইপারলিংক (গভীর বিশ্লেষণের জন্য)
```

## 💼 কমপ্লিট ড্যাশবোর্ড উদাহরণ

### টাইটেল: "বার্ষিক ফাইন্যান্সিয়াল ড্যাশবোর্ড ২০২৪"

```excel
সারি ১ (হেডার):
┌──────────────────────────────────────────────┐
│  📊 ফাইন্যান্সিয়াল ড্যাশবোর্ড          ২০২৪│
└──────────────────────────────────────────────┘

সারি ২ (KPI):
┌──────────┬──────────┬──────────┬──────────────┐
│ মোট রাজস্ব│ মোট খরচ │ নিট লাভ  │ লাভ মার্জিন  │
│ $২.৫M   │ $১.৮M   │ $০.৭M   │ ২৮% ↑       │
│ ↑ ১৫%   │ ↑ ৮%    │ ↑ ২৫%   │ YoY:+৩%     │
└──────────┴──────────┴──────────┴──────────────┘

সারি ৩ (প্রধান চার্ট):
┌──────────────────────────────────────────────┐
│   রাজস্ব ও খরচ ট্রেন্ড (মাসিক)              │
│   [লাইন চার্ট - রাজস্ব (নীল), খরচ (লাল)]      │
│   টার্গেট লাইন (ড্যাশড)                       │

সারি ৪ (সাপোর্টিং):

│  [পাই চার্ট] │ [বার চার্ট] │  [টেবিল]          │
│             │             │  স্পার্কলাইন সহ   │
└─────────────┴─────────────┴──────────────────┘

সারি ৫ (ফিল্টার):
┌──────────────────────────────────────────────┐
│ [বছর] [অঞ্চল] [পণ্য ক্যাটাগরি] [স্লাইসার]    │

```

   - ৪টি KPI কার্ড
   - ১টি লাইন চার্ট (মাসিক ট্রেন্ড)
   - ১টি পাই চার্ট (বিভাগ)
   - ১টি বার চার্ট (Top 5)
   - ১টি ডেটা টেবিল (স্পার্কলাইন সহ)
   - স্লাইসার ও টাইমলাইন

## 📝 ড্যাশবোর্ড টিপস
- **Alt + Enter**: সেলের ভিতরে নিউ লাইন
- **Camera Tool**: চার্ট/রেঞ্জের লাইভ ইমেজ
- **Transparent Shapes**: ডিজাইন এলিমেন্টের জন্য
- **Named Ranges**: ফর্মুলা সহজ রাখে
- **Hide Gridlines**: View → Uncheck Gridlines
- **Print Setup**: এক পৃষ্ঠায় ফিট করুন
- **ফন্ট**: KPI-র জন্য বড় (২৪-৩৬pt), লেবেলের জন্য ছোট (১০-১২pt)
- **অ্যানিমেশন নয়** — স্ট্যাটিক কিন্তু আপডেটেবল রাখুন
