# দিন ৪: XLOOKUP ও নেস্টেড লুকআপ (XLOOKUP & Nested Lookups)

## 🎯 আজকের লক্ষ্য
Excel-এর নতুন XLOOKUP ফাংশন এবং জটিল নেস্টেড লুকআপ কৌশল শেখা।

## 🚀 XLOOKUP ফাংশন

### সিনট্যাক্স
```excel
=XLOOKUP(লুকআপ_ভ্যালু, লুকআপ_অ্যারে, রিটার্ন_অ্যারে, [not_found], [match_mode], [search_mode])
```

### প্যারামিটার ব্যাখ্যা
| প্যারামিটার | বর্ণনা | ডিফল্ট |
|-------------|--------|---------|
| lookup_value | যা খুঁজবেন | - |
| lookup_array | যেখানে খুঁজবেন | - |
| return_array | যেখান থেকে মান আনবেন | - |
| not_found | না পেলে কী দেখাবে | #N/A |
| match_mode | 0=exact, -1=approx, 1=approx, 2=wildcard | 0 |
| search_mode | 1=first, -1=last, 2=binary, -2=binary | 1 |

### VLOOKUP-এর চেয়ে সুবিধা
- ✅ বাম বা ডান, উভয় দিকেই খুঁজতে পারে
- ✅ বিল্ট-ইন এরর হ্যান্ডলিং
- ✅ একক ফর্মুলা, কলাম নম্বর লাগে না
- ✅ কলাম যোগ/মুছে ফেললে ভুল হয় না
- ✅ ডিফল্ট exact match (VLOOKUP-এ ডিফল্ট approximate)

### ফাইন্যান্স উদাহরণ
```excel
// সহজ ব্যবহার
=XLOOKUP("P1001", A2:A100, D2:D100)

// এরর হ্যান্ডলিং সহ
=XLOOKUP("P1001", A2:A100, D2:D100, "পণ্য পাওয়া যায়নি")

// শেষ ম্যাচ খোঁজা (সর্বশেষ লেনদেন)
=XLOOKUP("P1001", A2:A100, D2:D100, "নেই", 0, -1)

// ওয়াইল্ডকার্ড সার্চ
=XLOOKUP("*ল্যাপটপ*", A2:A100, D2:D100, "নেই", 2)
```

## 🔄 নেস্টেড লুকআপ (Nested Lookups)

### এক লুকআপের ভিতরে আরেক লুকআপ
```excel
// প্রথমে পণ্য আইডি খুঁজে, তারপর দাম বের করা
=XLOOKUP(XLOOKUP(F2, পণ্য_আইডি, পণ্য_নাম), দাম_টেবিল_নাম, দাম_কলাম)
```

### দুই-মাত্রিক লুকআপ (Two-Way Lookup)
```excel
// সারি ও কলাম দুটোই ডায়নামিক
=XLOOKUP(H2, সারি_হেডার, XLOOKUP(I2, কলাম_হেডার, ডেটা_ম্যাট্রিক্স))
```

## 💼 ফাইন্যান্স কেস স্টাডি

### পরিস্থিতি ১: দামের গ্রেডেশন
```excel
// ভলিউম অনুযায়ী ডিসকাউন্ট রেট বের করা
=XLOOKUP(পরিমাণ, {0,100,500,1000}, {0,0.05,0.10,0.15}, "নেই", -1)
// -1 = exact match or next smaller item
```

### পরিস্থিতি ২: একাধিক শর্তের লুকআপ
```excel
// বছর ও মাস অনুযায়ী বিক্রয় বের করা
=XLOOKUP(1, (বছর_কলাম=২০২৪)*(মাস_কলাম="জানুয়ারি"), বিক্রয়_কলাম)
// '*' = AND অপারেটর
```

### পরিস্থিতি ৩: পণ্য শ্রেণিবিন্যাস
```excel
// বিভাগ → সাব-বিভাগ → পণ্য
=XLOOKUP(
    XLOOKUP(পণ্য_কোড, পণ্য_টেবিল[কোড], পণ্য_টেবিল[বিভাগ]),
    বিভাগ_টেবিল[বিভাগ_নাম],
    বিভাগ_টেবিল[বিভাগ_প্রধান]
)
```

## 🔧 XLOOKUP অ্যাডভান্সড প্যাটার্ন

### 1. ডায়নামিক রেঞ্জ
```excel
=XLOOKUP(F2, টেবিল1[কোড], টেবিল1[দাম])
// টেবিল রেফারেন্স ব্যবহার করলে ডেটা বাড়লেও কাজ করে
```

### 2. কন্ডিশনাল লুকআপ
```excel
// যদি পণ্য থাকে, তবে দাম দেখাও; নাহলে "স্টকে নেই"
=IF(ISNUMBER(XLOOKUP(F2, A:A, B:B)), XLOOKUP(F2, A:A, B:B), "স্টকে নেই")
// সরাসরি XLOOKUP-এ not_found ব্যবহার করাই ভালো
```

### 3. সর্বশেষ নন-ব্ল্যাংক মান
```excel
=XLOOKUP(TRUE, B:B<>"", B:B, "", 0, -1)
// শেষ ভর্তি সেলের মান বের করে
```

## 🆚 XLOOKUP vs VLOOKUP vs INDEX-MATCH

| বৈশিষ্ট্য | VLOOKUP | INDEX-MATCH | XLOOKUP |
|-----------|---------|-------------|---------|
| বাম দিকে খোঁজা | ❌ | ✅ | ✅ |
| ডান দিকে খোঁজা | ✅ | ✅ | ✅ |
| বিল্ট-ইন এরর হ্যান্ডলিং | ❌ | ❌ | ✅ |
| দুই-মাত্রিক লুকআপ | ❌ | ✅(জটিল) | ✅(সহজ) |
| ওয়াইল্ডকার্ড | ✅ | ❌ | ✅ |
| কলাম নিরাপদ | ❌ | ✅ | ✅ |
| ব্যবহার সহজতর | ✅ | ❌ | ✅✅ |

## ✅ প্র্যাকটিস টাস্ক
1. XLOOKUP ব্যবহার করে:
   - ইম্প্লয়ি আইডি দিয়ে নাম ও স্যালারি বের করুন
   - না পেলে "ডেটা পাওয়া যায়নি" দেখান
2. দুই-মাত্রিক লুকআপ তৈরি করুন:
   - সারিতে পণ্য, কলামে মাস — ভিতরে বিক্রয়
3. নেস্টেড লুকআপ তৈরি করুন:
   - এক টেবিল থেকে অন্য টেবিলে তথ্য আনুন
4. ডিসকাউন্ট স্ল্যাব তৈরি করুন XLOOKUP দিয়ে (পরিমাণ অনুযায়ী %)

> **পরবর্তী দিন:** কন্ডিশনাল ফরম্যাটিং ও ডেটা ভ্যালিডেশন!
