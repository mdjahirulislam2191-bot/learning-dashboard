# দিন ৭: What-If Analysis (গোল সিক, সিনারিও ম্যানেজার)

## 🎯 আজকের লক্ষ্য
Excel-এর What-If Analysis টুল ব্যবহার করে ভবিষ্যদ্বাণী, লক্ষ্য নির্ধারণ ও সিনারিও বিশ্লেষণ শেখা।

## 🎯 ১. গোল সিক (Goal Seek)

### গোল সিক কী?
যখন আপনি জানেন কী ফলাফল চান, কিন্তু কোন ইনপুট দিলে তা হবে — জানেন না।

### সিনট্যাক্স
```
Data → What-If Analysis → Goal Seek
```

### প্যারামিটার
| প্যারামিটার | ব্যাখ্যা |
|-------------|---------|
| Set cell | যে সেলে ফলাফল চান (ফর্মুলা থাকতে হবে) |
| To value | কাঙ্ক্ষিত ফলাফল |
| By changing cell | যে ইনপুট পরিবর্তন হবে |

### 💼 ফাইন্যান্স উদাহরণ

#### উদাহরণ ১: লোন পেমেন্ট ক্যালকুলেশন
```
প্রশ্ন: ৫০,০০০ ডলার লোনের মাসিক পেমেন্ট ১০০০ হলে
        সুদের হার কত হবে?

সেটআপ:
- B1: লোনের পরিমাণ = ৫০০০০
- B2: বার্ষিক সুদ = ৫% (পরিবর্তন হবে)
- B3: মাসিক পেমেন্ট = PMT(B2/12, 60, B1)
- গোল সিক: Set B3=1000, Change B2
```

#### উদাহরণ ২: ব্রেক-ইভেন পয়েন্ট
```
প্রশ্ন: কত ইউনিট বিক্রি করলে ব্রেক-ইভেন হবে?

সেটআপ:
- B1: ইউনিট বিক্রয় = ১০০ (পরিবর্তন হবে)
- B2: ইউনিট মূল্য = ৫০
- B3: ফিক্সড খরচ = ১০০০০
- B4: ভেরিয়েবল খরচ/ইউনিট = ৩০
- B5: লাভ = B1*(B2-B4) - B3
- গোল সিক: Set B5=0, Change B1
```

#### উদাহরণ ৩: স্যালস টার্গেট
```
প্রশ্ন: ২০% লাভ মার্জিন পেতে কত বিক্রয় দরকার?

সেটআপ:
- B1: বিক্রয় = ১০০০০০ (পরিবর্তন হবে)
- B2: খরচ = ৭৫০০০
- B3: লাভ = B1-B2
- B4: মার্জিন = B3/B1
- গোল সিক: Set B4=0.20, Change B1
```

## 📋 ২. ডেটা টেবিল (Data Table)

### ওয়ান-ওয়ে ডেটা টেবিল
একটি ইনপুট পরিবর্তন করে বিভিন্ন ফলাফল দেখা:
```excel
বিভিন্ন সুদের হারে মাসিক পেমেন্ট:
সুদের হার → ৪%, ৫%, ৬%, ৭%, ৮%
ফলাফল   → সংশ্লিষ্ট মাসিক পেমেন্ট
```

### টু-ওয়ে ডেটা টেবিল
দুইটি ইনপুট পরিবর্তন করে ফলাফল দেখা:
```excel
সুদের হার ও লোনের মেয়াদ → মাসিক পেমেন্ট ম্যাট্রিক্স

         | ৩ বছর | ৫ বছর | ১০ বছর
৪%       | xxx   | xxx   | xxx
৫%       | xxx   | xxx   | xxx
৬%       | xxx   | xxx   | xxx
```

## 📊 ৩. সিনারিও ম্যানেজার (Scenario Manager)

### সিনারিও ম্যানেজার কী?
বিভিন্ন "কী-যদি" পরিস্থিতি তৈরি ও তুলনা করা — একসাথে একাধিক ইনপুট পরিবর্তন করে।

### তৈরি করার ধাপ
```
1. Data → What-If Analysis → Scenario Manager
2. Add → Scenario Name দিন
3. Changing Cells সিলেক্ট করুন
4. প্রতিটি সিনারিওর জন্য মান দিন
5. Add → আরও সিনারিও যোগ করুন
```

### 💼 ফাইন্যান্স উদাহরণ

#### সিনারিও: বার্ষিক বাজেট পরিকল্পনা
```excel
সিনারিও ১: "বেস্ট কেস"
- বিক্রয় বৃদ্ধি: ২০%
- খরচ বৃদ্ধি: ৫%
- নিট লাভ: ?

সিনারিও ২: "ওয়ার্স্ট কেস"
- বিক্রয় বৃদ্ধি: -৫%
- খরচ বৃদ্ধি: ১৫%
- নিট লাভ: ?

সিনারিও ৩: "মোস্ট লাইকলি"
- বিক্রয় বৃদ্ধি: ১০%
- খরচ বৃদ্ধি: ৮%
- নিট লাভ: ?
```

### সিনারিও সামারি রিপোর্ট
```
Scenario Manager → Summary → 
Result cells সিলেক্ট করুন
Excel অটো একটি নতুন শীটে সব সিনারিওর তুলনা তৈরি করবে
```

## 🧩 ৪. সলভার (Solver) - পরিচিতি

### সলভার কী?
একাধিক সীমাবদ্ধতার মধ্যে সেরা সমাধান খুঁজে বের করা।

### কখন ব্যবহার করবেন?
- বাজেট বরাদ্দ অপটিমাইজ
- প্রোডাক্ট মিক্স ডিসিশন
- রিসোর্স এলোকেশন
- ইনভেস্টমেন্ট পোর্টফোলিও

### ফাইন্যান্স উদাহরণ:
```excel
প্রশ্ন: সীমিত বাজেটে কোন পণ্যে কত বিনিয়োগ করলে 
        সর্বোচ্চ লাভ হবে?

সীমাবদ্ধতা:
- মোট বাজেট ≤ ১০০০০০
- প্রতিটি পণ্যে ন্যূনতম ৫০০০
- রিস্ক স্কোর ≤ মধ্যম
```

## 💼 ডিটেইলড ফাইন্যান্স কেস স্টাডি

### কেস: লোন অ্যাফোর্ডেবিলিটি অ্যানালাইসিস
```excel
পরিস্থিতি:
আপনি ৩০,০০০ ডলার আয়ের চাকরি পেয়েছেন।
একটি ২৫০,০০০ ডলারের বাড়ির জন্য লোন নিতে চান।

গোল সিক ব্যবহার:
১. কত ডাউন পেমেন্ট দিলে মাসিক পেমেন্ট 
   ৩০% আয়ের মধ্যে থাকবে?
   
২. বিভিন্ন সুদের হারে (৪-৭%) মাসিক পেমেন্ট 
   কত হবে? (ডেটা টেবিল)

৩. তিনটি সিনারিও তৈরি করুন:
   - সুদ বাড়লে (৭%)
   - সুদ কমলে (৪%)
   - বর্তমান অবস্থা (৫.৫%)
```

## ✅ প্র্যাকটিস টাস্ক
1. গোল সিক ব্যবহার করে ব্রেক-ইভেন পয়েন্ট বের করুন
2. একটি লোন ক্যালকুলেটর তৈরি করুন (PMT ফাংশন সহ)
3. ডেটা টেবিল দিয়ে বিভিন্ন সুদের হারে পেমেন্ট দেখান
4. ৩টি সিনারিও তৈরি করুন (Best/Worst/Most Likely)
5. সিনারিও সামারি রিপোর্ট জেনারেট করুন
6. সলভারের ইন্টারফেস দেখে নিন

## 📝 টিপস
- গোল সিক শুধু একটি সেল পরিবর্তন করে — জটিল সমস্যার জন্য সলভার
- সিনারিও রিপোর্ট **PivotTable** ফরম্যাটেও তৈরি করা যায়
- What-If Analysis টুলগুলো **সিদ্ধান্ত গ্রহণে** সাহায্য করে, ভবিষ্যদ্বাণী নয়
- ডেটা টেবিল বড় ফাইল স্লো করতে পারে — শুধু প্রয়োজনীয় হিসেব রাখুন

> **পরবর্তী দিন:** Power Query — ডেটা ইম্পোর্ট ও ক্লিনিং!
