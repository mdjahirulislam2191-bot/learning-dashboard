# দিন ৫: কন্ডিশনাল ফরম্যাটিং ও ডেটা ভ্যালিডেশন

## 🎯 আজকের লক্ষ্য
ডেটাকে ভিজুয়ালি আকর্ষণীয় এবং নির্ভুল করার জন্য কন্ডিশনাল ফরম্যাটিং ও ডেটা ভ্যালিডেশন শেখা।

# 📌 পার্ট ১: কন্ডিশনাল ফরম্যাটিং (Conditional Formatting)

## কন্ডিশনাল ফরম্যাটিং কী?
নির্দিষ্ট শর্ত পূরণ করলে সেল/রেঞ্জের ফরম্যাট (কালার, ফন্ট, বর্ডার) অটোমেটিক পরিবর্তন।

## 🎨 প্রকারভেদ

### ১. হাইলাইট সেল রুলস (Highlight Cells Rules)
```excel
Greater Than...    → ১০,০০০ এর বেশি বিক্রয় সবুজ
Less Than...       → ৫,০০০ এর কম বিক্রয় লাল
Between...         → ১০,০০০-২০,০০০ এর মধ্যে
Equal To...        → নির্দিষ্ট মান
Text That Contains... → নির্দিষ্ট টেক্সট
A Date Occurring...   → আজ, গতকাল, এই সপ্তাহ
Duplicate Values...   → ডুপ্লিকেট চিহ্নিত
```

### ২. টপ/বটম রুলস (Top/Bottom Rules)
```
Top 10 Items       → সর্বোচ্চ ১০টি
Top 10%            → সর্বোচ্চ ১০%
Bottom 10 Items    → সর্বনিম্ন ১০টি
Above Average      → গড়ের উপরে
Below Average      → গড়ের নিচে
```

### ৩. ডেটা বার (Data Bars)
- সেলের ভিতরে গ্রেডিয়েন্ট বার দেখায়
- বড় মান = লম্বা বার
- ফাইন্যান্সে ব্যবহার: বিক্রয়, খরচ, লাভের তুলনা

### ৪. কালার স্কেল (Color Scales)
- ২-কালার বা ৩-কালার গ্রেডিয়েন্ট
- যেমন: লাল (নিম্ন) → হলুদ (মধ্যম) → সবুজ (উচ্চ)
- রিস্ক অ্যাসেসমেন্টের জন্য উপযোগী

### ৫. আইকন সেট (Icon Sets)
```
⬆️ 🟡 ⬇️  → ট্রেন্ড নির্দেশক
✅ ⚠️ ❌  → স্টেটাস নির্দেশক
🟢 🟡 🔴 → পারফরম্যান্স নির্দেশক
```

## 💼 ফাইন্যান্স উদাহরণ

### উদাহরণ ১: অ্যাকাউন্ট রিসিভেবল এজিং
```excel
শর্ত:           ফরম্যাট:
০-৩০ দিন        🟢 সবুজ
৩১-৬০ দিন       🟡 হলুদ  
৬১-৯০ দিন       🟠 কমলা
৯০+ দিন          🔴 লাল
```

### উদাহরণ ২: বাজেট ভ্যারিয়েন্স
```excel
ফর্মুলা: =C2>B2 (প্রকৃত > বাজেট)
ফরম্যাট: লাল ব্যাকগ্রাউন্ড (ওভার বাজেট)

ফর্মুলা: =C2<B2*0.9 (প্রকৃত < বাজেটের ৯০%)
ফরম্যাট: সবুজ ব্যাকগ্রাউন্ড (আন্ডার বাজেট)
```

### উদাহরণ ৩: ডুপ্লিকেট ইনভয়েস চিহ্নিত
```excel
Home → Conditional Formatting → Highlight Cell Rules → Duplicate Values
```

## 📋 পার্ট ২: ডেটা ভ্যালিডেশন (Data Validation)

### ডেটা ভ্যালিডেশন কী?
সেলে কী ধরনের ডেটা প্রবেশ করানো যাবে তা নিয়ন্ত্রণ করা — ডেটার নির্ভুলতা বাড়ায়।

### অ্যালাউড অপশনসমূহ

#### ১. Any Value (ডিফল্ট)
#### ২. Whole Number
```excel
যেমন: পরিমাণ ১-১০০০ এর মধ্যে
Data: between → Minimum: 1 → Maximum: 1000
```

#### ৩. Decimal
```excel
যেমন: মূল্য ০.০১ - ৯৯৯৯৯.৯৯
Data: between → Minimum: 0.01 → Maximum: 99999.99
```

#### ৪. List (ড্রপডাউন)
```excel
Source: পণ্য, সেবা, অন্যান্য
অথবা: =$A$2:$A$10 (রেঞ্জ থেকে)
```

#### ৫. Date / Time
```excel
যেমন: ভবিষ্যতের তারিখ নেই
Data: less than or equal to → =TODAY()
```

#### ৬. Text Length
```excel
যেমন: পোস্টাল কোড ৪ ডিজিট
Data: equal to → 4
```

#### ৭. Custom (ফর্মুলা ভিত্তিক)
```excel
=ISNUMBER(A2)              → শুধু সংখ্যা
=AND(A2>=0, A2<=100)       → ০-১০০ এর মধ্যে
=COUNTIF($A$2:$A$100, A2)=1 → ইউনিক ভ্যালু
```

### ইনপুট মেসেজ ও এরর অ্যালার্ট
```
Input Message: "শুধু পণ্য কোড P- ফরম্যাটে দিন"
Error Alert: Stop → "ভুল ফরম্যাট! P1001 এর মতো করে দিন"
```

## 💼 ফাইন্যান্সে ডেটা ভ্যালিডেশন

### উদাহরণ ১: লেনদেন ফর্ম
```excel
ক্ষেত্র:           ভ্যালিডেশন:
লেনদেন টাইপ      → ড্রপডাউন (ইনকাম, এক্সপেন্স, ট্রান্সফার)
পরিমাণ           → >০ (পজিটিভ সংখ্যা)
তারিখ             → <=TODAY() (ভবিষ্যত তারিখ নয়)
বিবরণ            → টেক্সট, ২০০ ক্যারেক্টারের কম
```

### উদাহরণ ২: পণ্য এন্ট্রি
```excel
ক্ষেত্র:           ভ্যালিডেশন:
পণ্য কোড         → ইউনিক ভ্যালু কাস্টম ফর্মুলা
মূল্য             → >০ এবং ≤১০০০০০
স্টক পরিমাণ      → পূর্ণ সংখ্যা ≥০
ক্যাটাগরি        → ড্রপডাউন লিস্ট
```

## ✅ প্র্যাকটিস টাস্ক

### কন্ডিশনাল ফরম্যাটিং:
1. একটি বিক্রয় ডেটাসেটে:
   - ১০০০০+ বিক্রয় → সবুজ, <৫০০০ → লাল
   - Data Bars যোগ করুন
   - Top 5 আইটেম হাইলাইট করুন
   - Icon Sets দিয়ে ট্রেন্ড দেখান

### ডেটা ভ্যালিডেশন:
2. একটি লেনদেন ইনপুট ফর্ম তৈরি করুন:
   - টাইপের জন্য ড্রপডাউন
   - পরিমাণের জন্য রেঞ্জ
   - তারিখের জন্য ভ্যালিডেশন
   - কাস্টম ক্লায়েন্ট আইডি ফরম্যাট

## 📝 টিপস
- কন্ডিশনাল ফরম্যাটিং **Manage Rules** থেকে এডিট করা যায়
- ফরম্যাটিংয়ের অর্ডার গুরুত্বপূর্ণ — উপরের রুল প্রাধান্য পায়
- **Stop If True** চেক করে একাধিক রুল নিয়ন্ত্রণ
- ডেটা ভ্যালিডেশন কপি করতে **Copy → Paste Special → Validation**
- ভুল ডেটা বৃত্তাকারে দেখাতে **Data → Data Validation → Circle Invalid Data**

> **পরবর্তী দিন:** Charts & Sparklines — ডেটা ভিজুয়ালাইজেশন!
