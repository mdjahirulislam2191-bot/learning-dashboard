# দিন ২: পিভট টেবিল অ্যাডভান্সড (Advanced Pivot Tables)

## 🎯 আজকের লক্ষ্য
পিভট টেবিলের অ্যাডভান্সড ফিচার শেখা - স্লাইসার, টাইমলাইন, ফিল্টার ও কাস্টম ক্যালকুলেশন।

## 🔍 ফিল্টার অপশনসমূহ

### ১. রিপোর্ট ফিল্টার (Report Filter)
পিভট টেবিলের উপরে独立的 ফিল্টার তৈরি করে:
- একাধিক আইটেম সিলেক্ট করা
- ফিল্টার অনুযায়ী আলাদা শিটে রিপোর্ট দেখানো

### ২. লেবেল ফিল্টার (Label Filters)
```
Equals / Does Not Equal
Begins With / Ends With
Contains / Does Not Contain
Greater Than / Less Than
```

### ৩. ভ্যালু ফিল্টার (Value Filters)
```
Top 10 (সর্বোচ্চ বিক্রির ১০টি পণ্য)
Above Average (গড়ের উপরে)
Below Average (গড়ের নিচে)
```

## 🎛️ স্লাইসার (Slicers) - ভিজুয়াল ফিল্টার

### স্লাইসার তৈরি করা
```
PivotTable Analyze → Insert Slicer → Select Fields
```

### ফাইন্যান্স উদাহরণ:
``` 
বছরের স্লাইসার → ২০২৪ সিলেক্ট করলে শুধু ২০২৪র ডেটা
অঞ্চলের স্লাইসার → 'ঢাকা' সিলেক্ট করলে শুধু ঢাকার ডেটা
```

### স্লাইসারের সুবিধা:
- ✅ ভিজুয়াল, বোতামের মতো ইন্টারফেস
- ✅ একাধিক স্লাইসার একসাথে ব্যবহার
- ✅ কালার ও সাইজ কাস্টমাইজ করা যায়
- ✅ এক স্লাইসার একাধিক পিভট টেবিল কন্ট্রোল করতে পারে

## 📅 টাইমলাইন (Timelines)

### টাইমলাইন তৈরি
```
PivotTable Analyze → Insert Timeline → Date Field
```

### টাইমলাইন ফিচার:
- মাস, কোয়ার্টার, বছর অনুযায়ী ফিল্টার
- ড্র্যাগ করে সময় রেঞ্জ সিলেক্ট
- অ্যানিমেটেড ট্রানজিশন

## 🧮 ক্যালকুলেটেড ফিল্ড (Calculated Fields)

### নতুন ফিল্ড তৈরি
```
PivotTable Analyze → Fields, Items & Sets → Calculated Field
```

### ফাইন্যান্স উদাহরণ:
```excel
ফিল্ড নাম: লাভের_হার
ফর্মুলা: = ('বিক্রয়' - 'খরচ') / 'বিক্রয়' * 100
```

### অন্যান্য ক্যালকুলেশন:
- **লাভ মার্জিন**: =(বিক্রয়-ক্রয়_মূল্য)/বিক্রয়
- **গ্রোথ %**: =(বর্তমান_বছর-পূর্ববর্তী_বছর)/পূর্ববর্তী_বছর
- **কমিশন**: =বিক্রয়*০.০৫

## 📊 গ্র্যান্ড টোটাল ও সাবটোটাল

### গ্র্যান্ড টোটাল কাস্টমাইজ
```
Design → Grand Totals → On for Rows/Columns Only/Off
```

### সাবটোটাল অপশন
```
Design → Subtotals → Show at Bottom/Top/Do Not Show
```

## 🔄 GETPIVOTDATA ফর্মুলা
পিভট টেবিল থেকে নির্দিষ্ট ভ্যালু রেফারেন্স করতে:
```excel
=GETPIVOTDATA("বিক্রয়",$A$3,"পণ্য","ল্যাপটপ","অঞ্চল","ঢাকা")
```

## 🏢 ফাইন্যান্স কেস স্টাডি

### পরিস্থিতি: বার্ষিক বাজেট বিশ্লেষণ
```
প্রয়োজনীয়তা:
1. বিভাগভিত্তিক বাজেট vs প্রকৃত খরচ তুলনা
2. সর্বোচ্চ খরচের ৫টি ক্যাটাগরি চিহ্নিত
3. মাসিক খরচের প্রবণতা দেখা
4. লাভ মার্জিন ক্যালকুলেশন

সমাধান:
- পিভট টেবিলে বিভাগ → Rows, খরচ → Values
- ভ্যালু ফিল্টারে Top 5 ব্যবহার
- টাইমলাইন দিয়ে মাসিক ট্রেন্ড
- ক্যালকুলেটেড ফিল্ড দিয়ে লাভ মার্জিন
- স্লাইসার দিয়ে ইন্টারঅ্যাক্টিভ ফিল্টার
```

## ✅ প্র্যাকটিস টাস্ক
1. পূর্বের পিভট টেবিলে স্লাইসার যোগ করুন (অঞ্চল, পণ্য)
2. টাইমলাইন ইনসার্ট করুন
3. একটি ক্যালকুলেটেড ফিল্ড তৈরি করুন (লাভ মার্জিন)
4. একাধিক পিভট টেবিলে একই স্লাইসার কানেক্ট করুন
5. Top 5 বিক্রিত পণ্য দেখান

> **পরবর্তী দিন:** VLOOKUP, INDEX-MATCH - লুকআপ ফাংশন শিখব!
