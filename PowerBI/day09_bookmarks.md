# দিন ৯: বুকমার্ক ও নেভিগেশন (Bookmarks & Navigation) 📑

## 🎯 আজকে যা শিখবেন
- বুকমার্ক তৈরি ও ম্যানেজমেন্ট
- বাটন দিয়ে রিপোর্ট নেভিগেশন
- বুকমার্কের মাধ্যমে ভিউ টগল
- Selection Pane ব্যবহার
- ফিন্যান্স রিপোর্টে ইন্টারঅ্যাকটিভ ড্যাশবোর্ড

## 📚 তাত্ত্বিক ধারণা

### বুকমার্ক কী?
বুকমার্ক হলো একটি **স্ন্যাপশট** — এটি ভিজুয়ালের অবস্থা, ফিল্টার, স্লাইসার ও পেজের অবস্থা সংরক্ষণ করে। বুকমার্ক ব্যবহার করে আপনি:

- ✅ এক পেজেই একাধিক ভিউ তৈরি করতে পারেন
- ✅ ফিল্টারের অবস্থা সংরক্ষণ করতে পারেন
- ✅ ডায়নামিক স্টোরি তৈরি করতে পারেন
- ✅ ক্লিন ইউজার ইন্টারফেস তৈরি করতে পারেন

### বুকমার্কের প্রকারভেদ

| প্রকার | কখন ব্যবহার করবেন |
|--------|-------------------|
| **Personal** | নিজের রেফারেন্সের জন্য |
| **Report** | ইউজারদের দেখানোর জন্য |
| **Navigation** | পেজের মধ্যে যাওয়ার জন্য |

### বুকমার্কের উপাদান

| প্রপার্টি | ব্যাখ্যা |
|-----------|----------|
| Data | ফিল্টার, স্লাইসারের অবস্থা |
| Display | ভিজুয়ালের হাইড/শো অবস্থা |
| Current Page | কোন পেজে আছে |
| Visuals | কোন ভিজুয়াল নির্বাচিত |

## 💻 স্টেপ বাই স্টেপ: বুকমার্ক ও নেভিগেশন

### Step 1: বুকমার্ক প্যান খোলা
1. View → **Bookmarks Pane** ✅ চিহ্নিত করুন
2. Selection Pane-ও খুলুন (View → Selection Pane ✅)

### Step 2: প্রথম বুকমার্ক তৈরি
1. আপনার ফিন্যান্স রিপোর্টের **Overview Page**-এ যান
2. কিছু ফিল্টার সেট করুন (যেমন: শুধু 2024, শুধু Expense)
3. Bookmarks Pane → **Add** ক্লিক করুন
4. নাম দিন: "মাসিক ব্যয়ের সারাংশ"

### Step 3: দ্বিতীয় বুকমার্ক (ভিন্ন ভিউ)
1. স্লাইসার পরিবর্তন করুন (যেমন: Quarter 1)
2. আবার **Add** ক্লিক করুন
3. নাম দিন: "প্রথম কোয়ার্টার বিশ্লেষণ"

### Step 4: বুকমার্ক আপডেট ও ম্যানেজ

| অপারেশন | কীভাবে করবেন |
|----------|------------|
| **Rename** | বুকমার্কের নাম ডাবল-ক্লিক |
| **Update** | ভিউ পরিবর্তন → বুকমার্কের উপর ⋮ → Update |
| **Delete** | ⋮ → Delete |
| **Reorder** | ড্র্যাগ করে উপরে-নিচে |

### Step 5: বাটন নেভিগেশন তৈরি

এখন আমরা **বাটন** ব্যবহার করে বুকমার্ক নেভিগেশন তৈরি করব:

1. Insert → **Buttons** → **Bookmark Navigator**
2. Power BI অটো বাটন তৈরি করবে — কিন্তু আমরা কাস্টম বাটন বানাবো।

**কাস্টম বাটন তৈরি:**
1. Insert → **Blank Button**
2. Properties → **Action**
3. Type: **Bookmark**
4. Bookmark: আপনার তৈরি করা যেকোনো বুকমার্ক নির্বাচন করুন

### Step 6: বাটন ফরম্যাটিং

**বাটনের টেক্সট:**
- Button Text → Text: "মাসিক বিশ্লেষণ দেখুন"
- Font Size: 14pt
- Font Color: সাদা

**বাটনের ডিজাইন:**
- Shape: Rounded Rectangle (Corner Radius: 8px)
- Fill (Default): #0078D4 (নীল)
- Fill (On Hover): #106EBE (গাঢ় নীল)
- Fill (On Press): #005A9E (আরও গাঢ়)

**আইকন যোগ করুন:**
- Icon → Icon Type: Image (একটি চার্টের আইকন)
- অথবা Button টেক্সটের সাথে ইমোজি ব্যবহার করুন: 📊

### Step 7: Selection Pane দিয়ে ভিজুয়াল কন্ট্রোল

Selection Pane ব্যবহার করে আপনি ভিজুয়াল হাইড/শো করতে পারেন:

1. Selection Pane খুলুন (View → Selection Pane)
2. ভিজুয়ালের নামের পাশের আইকন ক্লিক করে Hide/Show করুন
3. বুকমার্কে এই অবস্থা সংরক্ষণ করতে ভুলবেন না

### Step 8: বুকমার্ক দিয়ে ভিউ টগল

**একই পেজে Expense vs Income টগল:**

1. দুটি চার্ট তৈরি করুন — একটি Expense, একটি Income — ওভারল্যাপ করে রাখুন
2. বুকমার্ক ১: Expense Chart দেখাচ্ছে, Income Chart লুকানো
3. বুকমার্ক ২: Income Chart দেখাচ্ছে, Expense Chart লুকানো
4. বাটনের Action → Bookmark → দুটি বুকমার্ক সাইকেল করুন

## 🧪 প্র্যাকটিস টাস্ক (Practice Tasks)

### টাস্ক ১: ফিন্যান্স নেভিগেশন সিস্টেম
তিনটি বাটন তৈরি করুন:
1. 📊 **Overview** → মাসিক মোট আয়-ব্যয়
2. 📈 **Trend** → টাইম সিরিজ ট্রেন্ড
3. 💰 **Budget** → বাজেট ভ্যারিয়েন্স

প্রতিটি বাটন আলাদা পেজে নিয়ে যাবে (Page Navigation → Page Name)।

### টাস্ক ২: Expense/Income টগল
একই পেজে একটি বাটন তৈরি করুন যা Expense View ও Income View-এর মধ্যে টগল করে।

### টাস্ক ৩: ডেটা হাইলাইট বুকমার্ক
একটি বুকমার্ক তৈরি করুন যেখানে:
- সবচেয়ে ব্যয়বহুল Top 3 ক্যাটাগরি হাইলাইটেড
- বাকিগুলো ডিম/ম্লান

(Selection Pane + Visual ফরম্যাটিং একসাথে ব্যবহার করুন)

### টাস্ক ৪: ক্লিন UI তৈরি
একটি ন্যাভিগেশন বার তৈরি করুন:
- বাটনগুলো এক লাইনে সাজানো
- Active বাটনের আলাদা কালার
- প্রতিটি পেজের জন্য আলাদা বাটন

### টাস্ক ৫: Auto-Page Refresh
Power BI Service-এ পাবলিশ করার জন্য **Page Refresh** বুকমার্ক তৈরি করুন (লাইভ ডেটার জন্য উপযোগী)।

## 💡 ফিন্যান্স টিপস

### বুকমার্কের Best Practices
1. ✅ **নামকরণ সিস্টেম:** `01_Overview`, `02_Details`, `03_Budget`
2. ✅ **প্রতিটি বুকমার্ক টেস্ট করুন** — কাজ করছে কিনা নিশ্চিত হোন
3. ✅ **ট্রানজিশন এফেক্ট:** বুকমার্কের মধ্যে Smooth Transition সেট করুন
4. ✅ **Mobile View:** বুকমার্ক মোবাইল লেআউটেও কাজ করে — টেস্ট করুন
5. ❌ **অপ্রয়োজনীয় বুকমার্ক:** ১০টির বেশি না রাখা ভালো

### সাধারণ ভুল
- ❌ ফিল্টার চেঞ্জ করে বুকমার্ক Update না করা
- ❌ Selection Pane ভুলে Visual Hide/Show না করা
- ❌ বাটনে Action লিংক না করা

### ফিন্যান্স নেভিগেশন — প্রস্তাবিত লেআউট

```
[🏠 Home]  [📊 Expense]  [💰 Budget]  [📈 Trends]  [⚙️ Settings]
```

প্রতিটি ট্যাব আলাদা পেজ → Power BI Page Navigator ব্যবহার করুন।
