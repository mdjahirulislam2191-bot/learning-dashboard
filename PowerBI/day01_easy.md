## 1. Power BI ki?

**Power BI = Microsoft er Data Visualization Tool**

Shohj bhashe: **Data theke shundor graph ar dashboard banano tool**.

```
Data (Excel/SQL)  ->  Power BI  ->  Shundor graph/Dashboard
                                          ->
                                    Bos ke dekhao

```

### Keno shikbo?

| Karon | Bhasha |
|-------|--------|
| ✅ Data Analyst der kaj e lage | 90% company babohar kore |
| ✅ Shohj | Drag & drop — komon code |
| ✅ Shundor | Interactive charts |
| ✅ Chakri pawar subidha | Job posting e chawa hoy |

## 2. Download & Install

### Step 1: Download koro

🔗 **https://powerbi.microsoft.com/desktop/**

> 💡 **Free** — kono taka lage na!

### Step 2: Install koro

1. Download kora `.exe` file open koro
2. "Next -> Next -> Install" click koro
3. 3-5 minute lagbe

### Step 3: Chalu koro

- Start Menu -> "Power BI Desktop" search koro
- Open koro
- Sign in koro (Microsoft account diye)

---

## 3. Interface porichiti

Power BI open korle ei jinishgulo dekbe:

| Angsho | Kaj |
|--------|-----|
| **Top Menu** | Home, View, Modeling |
| **Left Panel** | Fields (datar column) |
| **Center** | Visualization Area (graph banano jayga) |
| **Bottom Right** | Visualizations (graph er dhoron byekti kore) |

---

## 4. Prothom Chart banao - Step by Step

### Step 1: Data dhokao

1. Power BI open koro
2. **Home -> Enter Data** click koro
3. Ei table ta likho:

| Month | Income | Expense |
|-------|--------|---------|
| Jan | 4000 | 2500 |
| Feb | 4200 | 2600 |
| Mar | 3800 | 2400 |
| Apr | 4500 | 2700 |
| May | 4100 | 2500 |
| Jun | 4300 | 2600 |

4. Name dao: `finance_data`
5. **Load** click koro

### Step 2: Chart banao

1. **Visualizations** theke **Stacked Bar Chart** select koro
2. **Fields** theke:
   - `Month` -> **Axis** te tano
   - `Income` -> **Values** te tano
   - `Expense` -> **Values** te tano
3. Hoye gele! 🎉

### Step 3: Sajao

- **Format** (paint roller icon) -> Colors change koro
- **Title** jog koro: "Monthly Income vs Expense"

---

## 5. Arekta Chart - Pie Chart

1. **Pie Chart** select koro
2. `Month` -> **Legend**
3. `Income` -> **Values**
4. Kon month e koto aay dekho!
