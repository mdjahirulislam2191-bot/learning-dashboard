### সারসংক্ষেপ
সম্ভাবনা (Probability) অনিশ্চয়তার গাণিতিক ভাষা। Finance-এ এটি ঝুঁকি মূল্যায়ন, বিনিয়োগ সিদ্ধান্ত, এবং ভবিষ্যদ্বাণীর ভিত্তি।

## ১. সম্ভাবনার সংজ্ঞা

**সূত্র:**
```
P(Event) = অনুকূল ফলাফলের সংখ্যা / মোট সম্ভাব্য ফলাফলের সংখ্যা
```

**উদাহরণ:** একটি ন্যায্য মুদ্রা টসে Head পড়ার সম্ভাবনা = 1/2 = 0.5 বা 50%

---

## ২. গুরুত্বপূর্ণ Terminology

| Term | অর্থ | Finance উদাহরণ |
|------|------|----------------|
| Experiment | পরীক্ষা/প্রক্রিয়া | স্টকে বিনিয়োগ |
| Outcome | একটি সম্ভাব্য ফলাফল | লাভ/ক্ষতি |
| Event | এক বা একাধিক Outcome | স্টক ৫%↑ হওয়া |
| Sample Space | সব Outcomes-এর সেট | {লাভ, ক্ষতি, অপরিবর্তিত} |

---

## ৩. সম্ভাবনার তিনটি নিয়ম

### নিয়ম ১: P(কোনো ঘটনা) = 0 থেকে 1-এর মধ্যে
```
0 ≤ P(E) ≤ 1
```
- P(E) = 0 → ঘটনা কখনো ঘটবে না
- P(E) = 1 → ঘটনা নিশ্চিত

### নিয়ম ২: সব সম্ভাবনার যোগফল = 1
```
P(S) = 1  (S = Sample Space)
```

### নিয়ম ৩: পরিপূরক নিয়ম (Complement Rule)
```
P(না) = 1 - P(হ্যাঁ)
```

**Finance উদাহরণ:** যদি স্টকের দাম বাড়ার সম্ভাবনা ০.৬ হয়, তাহলে না বাড়ার সম্ভাবনা = 1 - 0.6 = 0.4

---

## ৪. যোগের সূত্র (Addition Rule)

### পারস্পরিক এক্সক্লুসিভ (Mutually Exclusive) ঘটনা:
```
P(A বা B) = P(A) + P(B)
```

**উদাহরণ:** একটি কোম্পানির লাভ বা লোকসানের সম্ভাবনা
- P(লাভ) = 0.4, P(ক্ষতি) = 0.3
- P(লাভ বা ক্ষতি) = 0.4 + 0.3 = 0.7

### পারস্পরিক অ-এক্সক্লুসিভ ঘটনা:
```
P(A বা B) = P(A) + P(B) - P(A এবং B)
```

---

## ৫. গুণের সূত্র (Multiplication Rule)

### স্বাধীন ঘটনা (Independent Events):
```
P(A এবং B) = P(A) × P(B)
```

**Finance উদাহরণ:** দুইটি ভিন্ন কোম্পানির শেয়ার দাম বাড়ার সম্ভাবনা (যদি স্বাধীন হয়)
- P(Apple↑) = 0.6, P(Tesla↑) = 0.5
- P(উভয়েই↑) = 0.6 × 0.5 = **0.3**

### নির্ভরশীল ঘটনা (Dependent Events):
```
P(A এবং B) = P(A) × P(B|A)
```
যেখানে P(B|A) = A ঘটলে B-এর সম্ভাবনা

---

## ৬. শর্তাধীন সম্ভাবনা (Conditional Probability)

**সূত্র:**
```
P(A|B) = P(A এবং B) / P(B)
```

**Finance উদাহরণ:** 
- P(বাজার ঊর্ধ্বমুখী) = 0.4
- P(সুদের হার কমবে) = 0.5
- P(বাজার↑ এবং সুদ↓) = 0.3

সুদের হার কমলে বাজার ঊর্ধ্বমুখী হওয়ার সম্ভাবনা:
```
P(বাজার↑ | সুদ↓) = 0.3 / 0.5 = 0.6
```

---

## ৭. বায়েসের থিওরেম (Bayes' Theorem)

**সূত্র:**
```
P(A|B) = P(B|A) × P(A) / P(B)
```

**Finance ব্যবহার:** নতুন তথ্য পাওয়ার পর সম্ভাবনা আপডেট করা।

**উদাহরণ:** 
- একটি স্টক ভালো কোম্পানি হওয়ার পূর্ব সম্ভাবনা P(ভালো) = 0.3
- ভালো কোম্পানির Q1 রিপোর্ট ভালো আসার সম্ভাবনা P(ভালোQ|ভালো) = 0.8
- খারাপ কোম্পানির Q1 রিপোর্ট ভালো আসার সম্ভাবনা P(ভালোQ|খারাপ) = 0.2
- নতুন Q1 রিপোর্ট ভালো আসলো — এখন P(ভালো|ভালোQ) কত?

```
P(ভালোQ) = P(ভালোQ|ভালো)×P(ভালো) + P(ভালোQ|খারাপ)×P(খারাপ)
         = 0.8×0.3 + 0.2×0.7 = 0.24 + 0.14 = 0.38

P(ভালো|ভালোQ) = (0.8 × 0.3) / 0.38 = 0.24 / 0.38 = 0.63
```
→ পূর্বের 0.3 থেকে আপডেট হয়ে **0.63**

## ✅ Python উদাহরণ

```python
# Probability calculations
p_up = 0.6
p_down = 0.4

# Both independent events
p_both_up = p_up * p_up
print(f"P(both up): {p_both_up}")

# Conditional probability
p_market_up = 0.4
p_rate_down = 0.5
p_market_up_and_rate_down = 0.3

p_market_up_given_rate_down = p_market_up_and_rate_down / p_rate_down
print(f"P(market up | rate down): {p_market_up_given_rate_down}")
```
