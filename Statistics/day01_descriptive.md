# দিন ১: বর্ণনামূলক পরিসংখ্যান (Descriptive Statistics)

## 📊 ডেটা Analyst / Finance-এ আবেদন

### সারসংক্ষেপ
বর্ণনামূলক পরিসংখ্যান ডেটাকে সংক্ষেপে বর্ণনা করতে ব্যবহৃত হয়। এটি ডেটার কেন্দ্রীয় প্রবণতা (Central Tendency) এবং বিস্তার (Dispersion) বোঝাতে সাহায্য করে।

---

## ১. গড় (Mean)

**সংজ্ঞা:** সব মানের যোগফলকে মান সংখ্যা দিয়ে ভাগ করলে যে মান পাওয়া যায়।

**সূত্র:** 
```
μ = (Σx) / n   (জনসংখ্যার জন্য)
x̄ = (Σx) / n   (নমুনার জন্য)
```

**Finance উদাহরণ:** 
5টি স্টকের দৈনিক রিটার্ন: [2%, 3%, -1%, 4%, 2%]
```
Mean = (2 + 3 + (-1) + 4 + 2) / 5 = 10 / 5 = 2%
```
→ গড় রিটার্ন **2%**

**ব্যবহার:** পোর্টফোলিওর গড় রিটার্ন, গড় খরচ, গড় বিক্রয়

**সীমাবদ্ধতা:** Outlier-এর প্রতি সংবেদনশীল (যেমন: 2, 3, -1, 4, 100 → গড় 21.6% যা বিভ্রান্তিকর)

---

## ২. মধ্যমা (Median)

**সংজ্ঞা:** ডেটাকে ক্রমানুসারে সাজালে যে মান ঠিক মাঝখানে থাকে।

**পদ্ধতি:**
- বিজোড় সংখ্যক ডেটা: মধ্যবর্তী মান
- জোড় সংখ্যক ডেটা: দুই মধ্যবর্তী মানের গড়

**Finance উদাহরণ:**
5টি কোম্পানির শেয়ার মূল্য: [50, 55, 60, 65, 500]
```
Median = 60  (তৃতীয় মান)
Mean = 146   (Outlier দ্বারা প্রভাবিত)
```
→ Median (60) বেশি নির্ভরযোগ্য

**ব্যবহার:** বাড়ির দামের মধ্যমা, বেতনের মধ্যমা — Outlier-প্রতিরোধী

---

## ৩. প্রচুরক (Mode)

**সংজ্ঞা:** ডেটাসেটে সবচেয়ে বেশি বার出现的 মান।

**Finance উদাহরণ:**
গ্রাহকের লেনদেনের পরিমাণ: [100, 100, 100, 200, 200, 300, 400]
```
Mode = 100 (3 বার)
```

**ব্যবহার:** সবচেয়ে সাধারণ লেনদেনের পরিমাণ, সর্বাধিক বিক্রিত পণ্য

---

## ৪. পাল্লা/ব্যাপ্তি (Range)

**সংজ্ঞা:** সর্বোচ্চ এবং সর্বনিম্ন মানের পার্থক্য।

**সূত্র:** Range = Maximum - Minimum

**Finance উদাহরণ:**
একটি স্টকের সাপ্তাহিক দাম: [100, 105, 98, 110, 102]
```
Range = 110 - 98 = 12
```

**ব্যবহার:** ঝুঁকির প্রাথমিক ধারণা, দামের ওঠানামা

**সীমাবদ্ধতা:** শুধু দুইটি মান বিবেচনা করে, সব ডেটা উপেক্ষা করে

---

## 📋 সারাংশ টেবিল

| পরিমাপ | বৈশিষ্ট্য | Outlier-প্রভাব | Finance ব্যবহার |
|---------|-----------|----------------|-----------------|
| Mean | সব মান ব্যবহার করে | উচ্চ | গড় রিটার্ন |
| Median | মধ্যবর্তী মান | নিম্ন | বাড়ির দাম |
| Mode | সবচেয়ে সাধারণ | নেই | লেনদেন বিশ্লেষণ |
| Range | বিস্তার | উচ্চ | ঝুঁকি পরিমাপ |

---

## ✅ Python কোড উদাহরণ

```python
import numpy as np

data = [2, 3, -1, 4, 2]

mean = np.mean(data)
median = np.median(data)
range_val = max(data) - min(data)

print(f"Mean: {mean}")
print(f"Median: {median}")
print(f"Range: {range_val}")
```
