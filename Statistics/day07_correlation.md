## ১. কোভ্যারিয়েন্স (Covariance)

**সংজ্ঞা:** দুটি ভেরিয়েবল একসাথে কতটা পরিবর্তিত হয় তা পরিমাপ করে।

**সূত্র (জনসংখ্যা):**
```
Cov(X, Y) = Σ(xi - x̄)(yi - ȳ) / n
```

**সূত্র (নমুনা):**
```
Cov(X, Y) = Σ(xi - x̄)(yi - ȳ) / (n - 1)
```

### ব্যাখ্যা:
| কোভ্যারিয়েন্স | অর্থ |
|---------------|------|
| ধনাত্মক (+) | X বাড়লে Y-ও বাড়ে |
| ঋণাত্মক (-) | X বাড়লে Y কমে |
| শূন্য (0) | কোনো রৈখিক সম্পর্ক নেই |

### Finance উদাহরণ:
স্টক A এবং স্টক B-এর মাসিক রিটার্ন:

| মাস | স্টক A | স্টক B |
|-----|--------|--------|
| 1 | 5% | 8% |
| 2 | 3% | 6% |
| 3 | -2% | -1% |
| 4 | 4% | 7% |

```
Cov(A, B) ≈ ধনাত্মক (একসাথে ঊর্ধ্বমুখী)
```

**সীমাবদ্ধতা:** কোভ্যারিয়েন্সের মানের একক ব্যাখ্যা করা কঠিন (নির্ভর করে ডেটার স্কেলের উপর)

## ২. কোরিলেশন (Correlation)

**সংজ্ঞা:** কোভ্যারিয়েন্সের স্ট্যান্ডার্ডাইজড রূপ যা -1 থেকে +1-এর মধ্যে থাকে।

**Pearson Correlation Coefficient (r):**
```
r = Cov(X, Y) / (σX × σY)
```

### ব্যাখ্যা:

| r-মান | সম্পর্ক | শক্তি |
|-------|---------|-------|
| +1.0 | সম্পূর্ণ ধনাত্মক | খুব শক্তিশালী |
| +0.7 to +0.9 | দৃঢ় ধনাত্মক | শক্তিশালী |
| +0.3 to +0.7 | মাঝারি ধনাত্মক | মাঝারি |
| 0 to +0.3 | দুর্বল ধনাত্মক | দুর্বল |
| 0 | কোনো সম্পর্ক নেই | নেই |
| -0.3 to 0 | দুর্বল ঋণাত্মক | দুর্বল |
| -0.7 to -0.3 | মাঝারি ঋণাত্মক | মাঝারি |
| -1.0 to -0.7 | দৃঢ় ঋণাত্মক | শক্তিশালী |
| -1.0 | সম্পূর্ণ ঋণাত্মক | খুব শক্তিশালী |

## ৩. Finance-এ কোরিলেশন

### পোর্টফোলিও ডাইভার্সিফিকেশন

**মূল নীতি:** কম কোরিলেশনের সম্পদ একত্রিত করলে পোর্টফোলিও ঝুঁকি কমে।

**পোর্টফোলিও ভ্যারিয়েন্স:**
```
σ²p = w₁²σ₁² + w₂²σ₂² + 2w₁w₂Cov(R₁, R₂)
```

**উদাহরণ:**

| জুটি | Correlation | পোর্টফোলিও ঝুঁকি |
|------|------------|------------------|
| স্টক + স্টক (একই সেক্টর) | +0.80 | উচ্চ |
| স্টক + বন্ড | -0.20 | নিম্ন |
| স্টক + গোল্ড | +0.05 | খুব নিম্ন |
| স্টক + স্টক (ভিন্ন সেক্টর) | +0.30 | মাঝারি |

### বাস্তব Finance কোরিলেশন:

| জুটি | সাধারণ Correlation |
|------|-------------------|
| S&P 500 - NASDAQ | +0.85 |
| US Stocks - US Bonds | -0.30 to +0.10 |
| US Stocks - Gold | +0.05 to +0.20 |
| US Stocks - Emerging Markets | +0.60 |
| Oil - Airline Stocks | -0.50 |

## ৪. স্পারম্যানের র‍্যাঙ্ক কোরিলেশন (Spearman's Rank Correlation)

**যখন ব্যবহার করবেন:** ডেটা Normal না হলে বা অর্ডিনাল ডেটার জন্য।

**সূত্র:**
```
ρ = 1 - (6 × Σd²) / (n(n² - 1))
```
যেখানে d = র‍্যাঙ্কের পার্থক্য

## ৫. কোরিলেশন ≠ কজেশন (Causation)

**গুরুত্বপূর্ণ সতর্কতা:** Correlation মানেই এক ভেরিয়েবল অন্যটির কারণ নয়।

**Finance উদাহরণ:**
- আইসক্রিম বিক্রি ↑ এবং ড্রাউনিং ↑ → Positive correlation
- আসল কারণ: গ্রীষ্ম (তাপমাত্রা)
- ভুল সিদ্ধান্ত: আইসক্রিম নিষিদ্ধ করলে ড্রাউনিং কমবে?

**Spurious Correlation-এর উদাহরণ:**
- "S&P 500 বৃদ্ধি এবং বাংলাদেশে চা উৎপাদন" — সহসম্পর্ক থাকতে পারে কিন্তু কার্যকারণ নেই

---

## ✅ Python উদাহরণ

```python
import numpy as np
import pandas as pd

# Sample stock returns
stock_a = [5, 3, -2, 4, 6, 1, -1, 3]
stock_b = [8, 6, -1, 7, 9, 3, -3, 5]

# Covariance
cov_matrix = np.cov(stock_a, stock_b)
cov = cov_matrix[0, 1]
print(f"Covariance: {cov:.2f}")

# Correlation
corr_matrix = np.corrcoef(stock_a, stock_b)
corr = corr_matrix[0, 1]
print(f"Correlation: {corr:.4f}")

# Using pandas
df = pd.DataFrame({'Stock_A': stock_a, 'Stock_B': stock_b})
print(f"\nPandas Correlation:\n{df.corr()}")

# Portfolio variance example
w1, w2 = 0.6, 0.4
var1, var2 = np.var(stock_a, ddof=1), np.var(stock_b, ddof=1)
cov_ab = cov

portfolio_var = w1**2*var1 + w2**2*var2 + 2*w1*w2*cov_ab
portfolio_std = np.sqrt(portfolio_var)
print(f"Portfolio Variance: {portfolio_var:.4f}")
print(f"Portfolio Std Dev: {portfolio_std:.4f}")
```

| পরিমাপ | পরিসর | একক | Finance ব্যবহার |
|--------|-------|------|----------------|
| Covariance | সীমাহীন | বর্গ একক | পোর্টফোলিও ভ্যারিয়েন্স |
