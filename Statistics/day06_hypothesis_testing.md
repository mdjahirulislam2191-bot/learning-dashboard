# দিন ৬: হাইপোথিসিস টেস্টিং — t-টেস্ট (Hypothesis Testing & t-tests)

## 📊 ডেটা Analyst / Finance-এ আবেদন

### সারসংক্ষেপ
হাইপোথিসিস টেস্টিং একটি ডেটা-চালিত সিদ্ধান্ত গ্রহণের প্রক্রিয়া। আমরা একটি দাবি (Hypothesis) পরীক্ষা করি এবং ডেটার ভিত্তিতে সেটি গ্রহণ বা বাতিল করি। t-টেস্ট ছোট নমুনার জন্য ব্যবহৃত হয়।

---

## ১. হাইপোথিসিস টেস্টিং-এর ধাপ

### Null Hypothesis (H₀) এবং Alternative Hypothesis (H₁)

| টাইপ | বিবরণ | উদাহরণ |
|------|--------|--------|
| H₀ (Null) | কোন পরিবর্তন নেই, কোন পার্থক্য নেই | নতুন কৌশলের রিটার্ন = পুরনো কৌশলের রিটার্ন |
| H₁ (Alternative) | পরিবর্তন আছে, পার্থক্য আছে | নতুন কৌশলের রিটার্ন ≠ পুরনো কৌশলের রিটার্ন |

### পাঁচটি ধাপ:
1. **Hypothesis গঠন:** H₀ এবং H₁ লেখা
2. **Significance Level (α) নির্বাচন:** সাধারণত 0.05
3. **Test Statistic গণনা:** t-value
4. **p-value নির্ণয়:** প্রমাণের শক্তি
5. **সিদ্ধান্ত:** p < α হলে H₀ বাতিল

---

## ২. Significance Level (α) এবং p-value

| Term | অর্থ |
|------|------|
| α (Alpha) | টাইপ I ত্রুটির সম্ভাবনা (ভুলভাবে H₀ বাতিল) |
| p-value | H₀ সত্য হলে এত চরম ফলাফল পাওয়ার সম্ভাবনা |
| Significance | p < α → Statistically significant |

**সাধারণ α মান:** 0.10, 0.05, 0.01

---

## ৩. One-Sample t-Test

**কখন ব্যবহার করবেন:** একটি নমুনার গড় একটি নির্দিষ্ট মানের সমান কিনা পরীক্ষা করতে।

**সূত্র:**
```
t = (x̄ - μ₀) / (s / √n)
```
যেখানে,
- x̄ = নমুনা গড়
- μ₀ = পরীক্ষিত মান
- s = নমুনা SD
- n = নমুনা আকার

**Finance উদাহরণ:**
একটি মিউচুয়াল ফান্ডের গড় রিটার্ন কি 10% থেকে আলাদা?

| বিবরণ | মান |
|-------|------|
| নমুনা গড় (x̄) | 12% |
| পরীক্ষিত মান (μ₀) | 10% |
| SD (s) | 5% |
| n | 25 |

```
t = (12 - 10) / (5 / √25) = 2 / 1 = 2.0
df = 24
```

**সিদ্ধান্ত:** α = 0.05, দুই-টেইলড টেস্টে critical value ≈ 2.064
- |t| = 2.0 < 2.064 → H₀ **গ্রহণ** (পার্থক্য প্রমাণিত নয়)

---

## ৪. Two-Sample t-Test (Independent)

**কখন ব্যবহার করবেন:** দুইটি স্বাধীন গ্রুপের গড় তুলনা করতে।

**সূত্র:**
```
t = (x̄₁ - x̄₂) / √(s₁²/n₁ + s₂²/n₂)
```

**Finance উদাহরণ:**
দুইটি ভিন্ন সেক্টরের স্টকের রিটার্ন কি আলাদা?

| | টেক সেক্টর | ব্যাংক সেক্টর |
|--|-----------|--------------|
| গড় রিটার্ন | 15% | 10% |
| SD | 8% | 6% |
| n | 30 | 30 |

```
t = (15 - 10) / √(64/30 + 36/30)
  = 5 / √(2.13 + 1.2)
  = 5 / √3.33
  = 5 / 1.82
  = 2.75
```

**সিদ্ধান্ত:** p < 0.05 → H₀ **বাতিল** → দুই সেক্টরের রিটার্নে উল্লেখযোগ্য পার্থক্য আছে

---

## ৫. Paired t-Test

**কখন ব্যবহার করবেন:** একই গ্রুপের before-after তুলনা (যেমন: একটি কৌশল প্রয়োগের আগে ও পরে)।

**সূত্র:**
```
t = d̄ / (sd / √n)
```
যেখানে d̄ = পার্থক্যের গড়, sd = পার্থক্যের SD

**Finance উদাহরণ:**
একটি ট্রেডিং কৌশল প্রয়োগের আগে ও পরে ১০টি স্টকের রিটার্ন:

| স্টক | আগে | পরে | পার্থক্য |
|------|-----|-----|---------|
| A | 5% | 8% | 3% |
| B | 3% | 7% | 4% |
| ... | | | |

```
d̄ = 2.5%, sd = 1.5%, n = 10
t = 2.5 / (1.5/√10) = 2.5 / 0.474 = 5.27
```

---

## ৬. টাইপ I এবং টাইপ II ত্রুটি

| | H₀ সত্য | H₀ মিথ্যা |
|--|---------|-----------|
| H₀ গ্রহণ | ✓ Correct | ❌ Type II Error (β) |
| H₀ বাতিল | ❌ Type I Error (α) | ✓ Correct Power |

**Finance-এ প্রভাব:**
- **Type I Error:** একটি খারাপ বিনিয়োগকে ভালো বলে ভুল করা — অর্থ_loss
- **Type II Error:** একটি ভালো বিনিয়োগকে খারাপ বলে সুযোগ হারানো

---

## ✅ Python উদাহরণ

```python
import scipy.stats as stats

# One-sample t-test
sample = [12, 14, 8, 9, 11, 15, 10, 13, 7, 11]
population_mean = 10

t_stat, p_value = stats.ttest_1samp(sample, population_mean)
print(f"One-sample t-test: t = {t_stat:.3f}, p = {p_value:.4f}")

# Two-sample t-test (independent)
tech = [12, 15, 18, 10, 8, 14, 20, 16, 11, 9]
bank = [8, 10, 12, 6, 9, 11, 7, 13, 10, 8]

t_stat, p_value = stats.ttest_ind(tech, bank)
print(f"Two-sample t-test: t = {t_stat:.3f}, p = {p_value:.4f}")

# Paired t-test
before = [5, 3, 6, 4, 7, 8, 5, 6, 4, 3]
after = [8, 7, 9, 6, 9, 10, 8, 9, 7, 8]

t_stat, p_value = stats.ttest_rel(before, after)
print(f"Paired t-test: t = {t_stat:.3f}, p = {p_value:.4f}")
```

| দুই গ্রুপের variance সমান? | Student's t-test |
