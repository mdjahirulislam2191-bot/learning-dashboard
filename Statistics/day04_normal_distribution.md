# দিন ৪: স্বাভাবিক বন্টন ও Z-স্কোর (Normal Distribution & Z-scores)

## 📊 ডেটা Analyst / Finance-এ আবেদন

### সারসংক্ষেপ
স্বাভাবিক বন্টন (Normal Distribution) পরিসংখ্যানের সবচেয়ে গুরুত্বপূর্ণ বন্টন। Finance-এ এটি স্টক রিটার্ন, ঝুঁকি মডেলিং, এবং মূল্য নির্ধারণে ব্যবহৃত হয়।

---

## ১. স্বাভাবিক বন্টনের বৈশিষ্ট্য

ঘণ্টার মতো আকৃতির বলে একে **Bell Curve**-ও বলা হয়।

**মূল বৈশিষ্ট্যসমূহ:**
- সমমিত (Symmetric) — Mean = Median = Mode
- ডেটার 68% Mean ± 1 SD-এর মধ্যে
- ডেটার 95% Mean ± 2 SD-এর মধ্যে
- ডেটার 99.7% Mean ± 3 SD-এর মধ্যে

**68-95-99.7 নিয়ম:**
```
μ ± 1σ → 68% ডেটা
μ ± 2σ → 95% ডেটা
μ ± 3σ → 99.7% ডেটা
```

---

## ২. Finance-এ Normal Distribution

**স্টক রিটার্নের উদাহরণ:**
- গড় রিটার্ন (μ) = 10%
- স্ট্যান্ডার্ড ডেভিয়েশন (σ) = 15%

```
68% সময় রিটার্ন: 10% ± 15% = [-5%, 25%]
95% সময় রিটার্ন: 10% ± 30% = [-20%, 40%]
99.7% সময় রিটার্ন: 10% ± 45% = [-35%, 55%]
```

---

## ৩. Z-স্কোর (Z-Score)

**সংজ্ঞা:** একটি ডেটা পয়েন্ট গড় থেকে কত SD দূরে তা প্রকাশ করে।

**সূত্র:**
```
Z = (x - μ) / σ
```

যেখানে,
- x = ডেটা পয়েন্ট
- μ = জনসংখ্যার গড়
- σ = জনসংখ্যার SD

### Z-স্কোরের ব্যাখ্যা:

| Z-স্কোর | অর্থ |
|---------|------|
| Z = 0 | গড়ের সমান |
| Z = 1 | গড় থেকে 1 SD উপরে |
| Z = -1 | গড় থেকে 1 SD নিচে |
| Z = 2 | গড় থেকে 2 SD উপরে |
| Z = -2 | গড় থেকে 2 SD নিচে |

---

## ৪. Finance উদাহরণ

ধরা যাক, S&P 500-এর দৈনিক রিটার্নের μ = 0.05%, σ = 1.2%।

**প্রশ্ন:** আজকের রিটার্ন 2.5% হলে Z-স্কোর কত?

```
Z = (2.5 - 0.05) / 1.2 = 2.45 / 1.2 = 2.04
```
→ আজকের রিটার্ন গড় থেকে 2.04 SD উপরে (অস্বাভাবিকভাবে ভালো দিন)

**প্রশ্ন:** -2% রিটার্নের Z-স্কোর কত?

```
Z = (-2 - 0.05) / 1.2 = -2.05 / 1.2 = -1.71
```
→ গড় থেকে 1.71 SD নিচে (খারাপ দিন, কিন্তু 95% 범위ের মধ্যে)

---

## ৫. Z-টেবিল (Z-Table) ব্যবহার

Z-টেবিল থেকে আমরা P(Z < নির্দিষ্ট মান) জানতে পারি।

**উদাহরণ:** Z = 1.5 হলে,
- P(Z < 1.5) = 0.9332
- অর্থাৎ 93.32% সময় Z-স্কোর 1.5-এর নিচে থাকে

**Finance ব্যবহার:** Value at Risk (VaR) 계산

**VaR উদাহরণ:** 95% আত্মবিশ্বাসে সর্বোচ্চ ক্ষতি কত?
```
Z for 95% = 1.645 (one-tailed)
VaR = μ - 1.645 × σ
```
যদি μ = 0.05%, σ = 1.2%:
```
VaR = 0.05% - 1.645 × 1.2% = 0.05% - 1.974% = -1.924%
```
→ 95% আত্মবিশ্বাসে দৈনিক ক্ষতি 1.924%-এর বেশি হবে না।

## ৬. স্কিউড বন্টন (Skewed Distribution)

বাস্তব Finance ডেটা প্রায়ই পুরোপুরি Normal হয় না:

| ধরন | বৈশিষ্ট্য | Finance উদাহরণ |
|------|-----------|----------------|
| Right-skewed | ডান লম্বা | স্টক রিটার্ন (কখনো কখনো বড় লাভ) |
| Left-skewed | বাম লম্বা | বন্ড রিটার্ন (নিচের ঝুঁকি) |
| Leptokurtic | মোটা লেজ | ক্রিপ্টো (অস্বাভাবিক বড় মুভমেন্ট) |

---

## ✅ Python উদাহরণ

```python
import numpy as np
import scipy.stats as stats

# Z-score calculation
mean = 10
std = 15
x = 25

z = (x - mean) / std
print(f"Z-score: {z:.2f}")

# Probability from Z-score
prob = stats.norm.cdf(z)
print(f"P(X < {x}): {prob:.4f}")

# Value at Risk (95% confidence)
z_95 = stats.norm.ppf(0.05)  # 5th percentile
var = mean + z_95 * std
print(f"VaR (95%): {var:.2f}")
```
