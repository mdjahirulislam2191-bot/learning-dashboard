# দিন ২: ভ্যারিয়েন্স ও স্ট্যান্ডার্ড ডেভিয়েশন (Variance & Standard Deviation)

## 📊 ডেটা Analyst / Finance-এ আবেদন

### সারসংক্ষেপ
ভ্যারিয়েন্স এবং স্ট্যান্ডার্ড ডেভিয়েশন ডেটার **বিক্ষিপ্ততা (Dispersion বা Spread)** পরিমাপ করে। Finance-এ এগুলি **ঝুঁকি (Risk)** মাপার প্রধান হাতিয়ার।

---

## ১. ভ্যারিয়েন্স (Variance)

**সংজ্ঞা:** প্রতিটি ডেটা পয়েন্টের গড় থেকে দূরত্বের বর্গের গড়।

**জনসংখ্যার ভ্যারিয়েন্স (σ²):**
```
σ² = Σ(xi - μ)² / N
```

**নমুনার ভ্যারিয়েন্স (s²):**
```
s² = Σ(xi - x̄)² / (n - 1)
```
> n-1 ব্যবহার করা হয় কারণ নমুনা জনসংখ্যার চেয়ে কম নির্ভুল — একে Bessel's correction বলে।

**ধাপে ধাপে উদাহরণ:**
স্টক রিটার্ন: [2%, 4%, 6%, 8%, 10%]

1. Mean = (2+4+6+8+10)/5 = 6%
2. Deviation: -4, -2, 0, 2, 4
3. Squared: 16, 4, 0, 4, 16
4. Variance = (16+4+0+4+16)/5 = 40/5 = **8**

---

## ২. স্ট্যান্ডার্ড ডেভিয়েশন (Standard Deviation)

**সংজ্ঞা:** ভ্যারিয়েন্সের বর্গমূল। এটি মূল ডেটার এককে থাকে, তাই ব্যাখ্যা করা সহজ।

**সূত্র:**
```
σ = √σ²    (জনসংখ্যা)
s = √s²    (নমুনা)
```

**উপরের উদাহরণ থেকে:**
```
σ = √8 = 2.83%
```
→ অর্থাৎ রিটার্ন সাধারণত গড় (6%) থেকে ±2.83%範圍ে ওঠানামা করে।

## ৩. Finance-এ ঝুঁকি পরিমাপ

### স্টক A: [5%, 5%, 5%, 5%, 5%]
- Mean = 5%, Variance = 0, SD = 0%
- **ঝুঁকি নেই** (সর্বদা একই রিটার্ন)

### স্টক B: [-10%, 20%, -5%, 15%, 0%]
- Mean = 4%, Variance = 162.5, SD = 12.7%
- **উচ্চ ঝুঁকি** (রিটার্ন অনিয়মিত)

### স্টক C: [2%, 4%, 3%, 5%, 6%]
- Mean = 4%, Variance = 2.5, SD = 1.58%
- **নিম্ন ঝুঁকি** (স্থিতিশীল রিটার্ন)

---

## ৪. Sharpe Ratio — রিস্ক-অ্যাডজাস্টেড রিটার্ন

```
Sharpe Ratio = (Rp - Rf) / σp
```
যেখানে,
- Rp = পোর্টফোলিও রিটার্ন
- Rf = ঝুঁকিমুক্ত রিটার্ন (যেমন: ট্রেজারি বন্ড)
- σp = পোর্টফোলিওর SD

**উদাহরণ:**
- পোর্টফোলিও রিটার্ন = 12%, Rf = 3%, SD = 8%
- Sharpe = (12-3)/8 = **1.125**

> উচ্চ Sharpe Ratio = ভালো রিস্ক-অ্যাডজাস্টেড পারফরম্যান্স

---

## ৫. Python উদাহরণ

```python
import numpy as np

returns = [2, 4, 6, 8, 10]

variance = np.var(returns, ddof=0)       # Population variance
variance_sample = np.var(returns, ddof=1) # Sample variance
std_dev = np.std(returns, ddof=0)

print(f"Variance (population): {variance}")
print(f"Variance (sample): {variance_sample}")
print(f"Standard Deviation: {std_dev:.2f}")
```

```python
# Sharpe Ratio
portfolio_return = 12
risk_free_rate = 3
std_dev = 8

sharpe = (portfolio_return - risk_free_rate) / std_dev
print(f"Sharpe Ratio: {sharpe:.2f}")
```
