## ১. স্যাম্পলিং (Sampling)

**জনসংখ্যা (Population):** পুরো গ্রুপ (যেমন: সব S&P 500 কোম্পানি)
**নমুনা (Sample):** জনসংখ্যার একটি উপসেট (যেমন: 50টি S&P 500 কোম্পানি)

### স্যাম্পলিং পদ্ধতি:

| পদ্ধতি | বর্ণনা | Finance ব্যবহার |
|--------|--------|-----------------|
| Simple Random | প্রতিটি সদস্যের সমান সম্ভাবনা | এলোমেলো স্টক নির্বাচন |
| Stratified | স্তরভিত্তিক (যেমন: সেক্টর অনুযায়ী) | সেক্টর অনুযায়ী পোর্টফোলিও |
| Cluster | ক্লাস্টারভিত্তিক | ভৌগোলিক অঞ্চল অনুযায়ী |
| Systematic | প্রতি k-তম সদস্য | সময়সিরিজ ডেটা |

## ২. স্যাম্পলিং এরর (Sampling Error)

**সংজ্ঞা:** নমুনা পরিসংখ্যান এবং জনসংখ্যার প্যারামিটারের মধ্যে পার্থক্য।

**সূত্র:** 
```
Sampling Error = x̄ - μ
```

**Standard Error (SE):** নমুনা গড়ের স্ট্যান্ডার্ড ডেভিয়েশন
```
SE = σ / √n
```

যেখানে n = নমুনার আকার

**Finance উদাহরণ:**
- জনসংখ্যার SD (σ) = 15%
- নমুনা আকার (n) = 100
- SE = 15 / √100 = 15/10 = **1.5%**

> নমুনার আকার বাড়ালে SE কমে → অনুমান বেশি নির্ভুল হয়

---

## ৩. কেন্দ্রীয় সীমা তত্ত্ব (Central Limit Theorem - CLT)

**গুরুত্বপূর্ণ তত্ত্ব:** নমুনার আকার যথেষ্ট বড় হলে (n ≥ 30), নমুনা গড়ের বন্টন প্রায় Normal হয় — মূল জনসংখ্যার বন্টন যাই হোক না কেন।

**CLT-এর সূত্র:**
```
x̄ ~ N(μ, σ/√n)
```

**Finance ব্যবহার:** 
- নমুনা ডেটা Normal না হলেও আমরা Normal Distribution ব্যবহার করতে পারি
- পোর্টফোলিও রিটার্নের কনফিডেন্স ইন্টারভ্যাল তৈরি

---

## ৪. কনফিডেন্স ইন্টারভ্যাল (Confidence Interval - CI)

**সংজ্ঞা:** একটি পরিসর যা নির্দিষ্ট আত্মবিশ্বাসের সাথে জনসংখ্যার প্যারামিটার ধারণ করে।

### সাধারণ সূত্র:
```
CI = x̄ ± (Critical Value × SE)
```

### 95% কনফিডেন্স ইন্টারভ্যাল:
```
CI = x̄ ± 1.96 × (σ / √n)
```

### Finance উদাহরণ:
একটি মিউচুয়াল ফান্ডের গড় রিটার্ন:
- x̄ = 12%
- σ = 8%
- n = 64

```
SE = 8 / √64 = 8/8 = 1%
CI = 12% ± 1.96 × 1% = 12% ± 1.96%
CI = [10.04%, 13.96%]
```

**ব্যাখ্যা:** 95% আত্মবিশ্বাসের সাথে বলা যায় যে প্রকৃত জনসংখ্যার গড় রিটার্ন 10.04% থেকে 13.96%-এর মধ্যে।

---

## ৫. বিভিন্ন Confidence Level-এর Critical Values

| Confidence Level | Alpha (α) | Critical Value (Z) |
|-----------------|-----------|-------------------|
| 90% | 0.10 | 1.645 |
| 95% | 0.05 | 1.960 |
| 99% | 0.01 | 2.576 |

---

## ৬. Margin of Error (ত্রুটির পরিধি)

**সূত্র:**
```
Margin of Error = Critical Value × SE
```

**উদাহরণ:** 95% CI-তে, Margin of Error = 1.96 × 1% = **1.96%**

---

## ৭. নমুনা আকার নির্ণয় (Sample Size Determination)

**সূত্র:**
```
n = (Z × σ / E)²
```
যেখানে E = কাঙ্ক্ষিত Margin of Error

**Finance উদাহরণ:**
- Z = 1.96 (95% confidence)
- σ = 10% (estimated)
- E = 2% (desired margin)

```
n = (1.96 × 10 / 2)² = (19.6/2)² = (9.8)² = 96.04 ≈ 97
```
→ 97টি নমুনা প্রয়োজন

---

## ✅ Python উদাহরণ

```python
import numpy as np
import scipy.stats as stats

# Sample data
returns = [10, 12, 8, 15, 9, 11, 14, 7, 13, 11]
n = len(returns)
mean = np.mean(returns)
se = stats.sem(returns)  # Standard Error

# 95% Confidence Interval
confidence = 0.95
ci = stats.t.interval(confidence, df=n-1, loc=mean, scale=se)

print(f"Sample Mean: {mean:.2f}%")
print(f"Standard Error: {se:.2f}%")
print(f"95% CI: {ci}")

# Required sample size
z = 1.96
sigma = 10
E = 2
n_required = (z * sigma / E) ** 2
print(f"Required sample size: {np.ceil(n_required)}")
```
